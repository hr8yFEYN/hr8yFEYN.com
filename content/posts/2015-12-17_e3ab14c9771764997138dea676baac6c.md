 <a href="https://farm1.staticflickr.com/946/42069249971_b0f678058f_o.jpg"><img alt="馬鞍底のマーケット" src="https://farm1.staticflickr.com/946/42069249971_4a6816432b_c.jpg"></a>
馬鞍底では日の干支が虎と猿の日にマーケットが開かれる。今日は虎の日なのでマーケット、市を見に行く。

雲南省は経度でいえば東南アジアと変わらないのだが時間は北京時間の為、朝7時過ぎにやっと明るくなってくる。窓から外の様子を見てみるがあまり人も歩いていないので、8時ごろまで待ってみる。まだ人通りは少ないがマーケットを見に行く。

<br>

<a href="https://farm1.staticflickr.com/947/42069419371_8e5dcb29b3_o.jpg"><img alt="馬鞍底のマーケット, 1" src="https://farm1.staticflickr.com/947/42069419371_33b78fb0a7_c.jpg"></a>
やはりまだまだ準備中といった感じだ。昨日からの雨で人の出足が悪いのかもともと朝が遅いマーケットなのか。しばらくぶらぶら歩いてみるがまだまだ人がまばらにしかいないので宿に戻って再度出かけることにする。

<br>

<a href="https://farm1.staticflickr.com/955/40262088130_41967b9751_o.jpg"><img alt="馬鞍底のマーケット, 2" src="https://farm1.staticflickr.com/955/40262088130_08fd9ba75e_c.jpg"></a>
10時ごろになったのでマーケットを見に行ってみる。先ほどよりは人がだいぶ多いがまだまだ人は少ない。もともとこの程度のマーケットなのかそれとも雨だから人出が少ないのか。

<br>

<a href="https://farm1.staticflickr.com/949/42023848682_3294127837_o.jpg"><img alt="馬鞍底のマーケット, 3" src="https://farm1.staticflickr.com/949/42023848682_a062c3b80f_c.jpg"></a>
きな粉もちが売っていた。以前、金平を旅行した時によく食べていた。実際はもちではなく白玉だが日本のきな粉もちのような味で本当においしい。

<br>

<a href="https://farm1.staticflickr.com/905/27199554487_c2701cfbc8_o.jpg"><img alt="馬鞍底のマーケット, 4" src="https://farm1.staticflickr.com/905/27199554487_ca3a9396d9_c.jpg"></a>
肉まんを揚げたものも売っていた。肉まんを揚げたものはベトナムではよく見かけるが中国ではあまり見かけない。馬鞍底はベトナムに近いからベトナムの影響かな。

<br>

<a href="https://farm1.staticflickr.com/825/41169334155_8f7140d377_o.jpg"><img alt="馬鞍底のマーケット, 5" src="https://farm1.staticflickr.com/825/41169334155_9d3e986bd9_c.jpg"></a>
雲南名物の水パイプ。むかしはバスにもパイプが積んであって客がゴボゴボゴボ、ボッカーと吸っていたが、中国でもタバコが吸えない場所が増えているらしいのでもうバスには積んでないかな。

<br>

<a href="https://farm1.staticflickr.com/972/41350279784_5c65e60dfe_o.jpg"><img alt="馬鞍底のマーケット, 6" src="https://farm1.staticflickr.com/972/41350279784_e4f0c5135d_c.jpg"></a>
こんにゃくが売っていた。雲南ではよく見かけるが、中国の北部ではあまり食べないらしい。こんにゃくは漢族の食べ物ではないく少数民族の食べ物なのかもしれない。となると日本人も少数民族の類だ。

<br>

<a href="https://farm1.staticflickr.com/982/41168844375_5741d1ac4b_o.jpg"><img alt="馬鞍底のマーケット, 7" src="https://farm1.staticflickr.com/982/41168844375_d67bf53395_c.jpg"></a>
漢方薬の露店で貝殻が売られていた。これは粉末にでもして服用するのだろうか。海は遠い、珍しいものはありがたい、ありがたいものは霊験もあらたかだ、ならば食べよう服用しようということなんだろう、おそらく。

<br>

<a href="https://farm1.staticflickr.com/910/28196830248_3f5fbd6ba5_o.jpg"><img alt="馬鞍底のマーケット, 8" src="https://farm1.staticflickr.com/910/28196830248_65478f6d28_c.jpg"></a>
ハニ族は大きなまげを載せている。

<br>

<a href="https://farm1.staticflickr.com/905/40261155180_0729423d74_o.jpg"><img alt="馬鞍底のマーケット, 9" src="https://farm1.staticflickr.com/905/40261155180_7d2b2caa05_c.jpg"></a>
まげが売っていた。

<br>

<a href="https://farm1.staticflickr.com/946/42024574842_f125b8ce7f_o.jpg"><img alt="馬鞍底のマーケット, 10" src="https://farm1.staticflickr.com/946/42024574842_f246ae08b7_c.jpg"></a>
ハニ族の子供がかぶる帽子。

<br>

<a href="https://farm1.staticflickr.com/960/42025309152_929712af70_o.jpg"><img alt="馬鞍底のマーケット, 11" src="https://farm1.staticflickr.com/960/42025309152_83903cef06_c.jpg"></a>
紅頭ヤオ族。この辺りはヤオ族も多いようだ。

民族衣装を着ている人は思っていたより少なかった。ハニ族が多く。次に紅頭ヤオ族が多い。数は少ないがプー族、ミャオ族もいた。昨日は民族衣装の上着を着たプー族を数人見かけたが、今日見かけたひとはスカートだけだった。

<br>
