 <a href="https://farm1.staticflickr.com/945/42069948031_d6d1a75cd8_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング" src="https://farm1.staticflickr.com/945/42069948031_4693f77d55_c.jpg"></a>
紅頭ヤオ族の村、板板橋に行く。まず昨日の昼に客がたくさん入っていた過橋米線の店に行って食事を済ませる。

<br>

<a href="https://farm1.staticflickr.com/954/42025080242_4ccbe4c543_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 1" src="https://farm1.staticflickr.com/954/42025080242_46858c3c51_c.jpg"></a>
一番安い8元の過橋米線を頼む。食券を持って厨房に行くと具を3つ選べるというので目玉焼き、豚肉と鶏肉をいれてもらう。米線は自分で食べる分だけ取って湯がくようだ。なので米線自体は食べ放題ということらしい。おいしかったのでまた明日も来ようと思う。

<br>

<a href="https://farm1.staticflickr.com/950/42023726292_312f8c7d18_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 2" src="https://farm1.staticflickr.com/950/42023726292_0572dcfb9e_c.jpg"></a>
食事も済んだので広場の脇から山の方に登って行く。久しぶりなので山道というか農道に出る道がちょっと思い出せなかったが、進んで行くと農道に出たのでこの道で合ってるだろうと進んでいく。

<br>

<a href="https://farm1.staticflickr.com/909/42024136342_57a6474d24_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 3" src="https://farm1.staticflickr.com/909/42024136342_bcc269a970_c.jpg"></a>
45分ほど歩くとハニ族の村に出る。この村には郷道が通っていて、その郷道を歩いて行けば板板村に着く。以前は未舗装路だったがいまはコンクリートで舗装されている。

<br>

<a href="https://farm1.staticflickr.com/910/42069825871_fa7603283e_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 4" src="https://farm1.staticflickr.com/910/42069825871_6519bef37a_c.jpg"></a>
3キロほど歩くと道の脇で紅頭ヤオ族のおばさん達がずいぶん大勢で何やら作業をしている。

<br>

<a href="https://farm1.staticflickr.com/981/42024097322_3e44ecf7d6_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 5" src="https://farm1.staticflickr.com/981/42024097322_155179e64f_c.jpg"></a>
1時間半ほどで板板村に着く。以前よりもだいぶコンクリートの家が増えている。

<br>

<a href="https://farm1.staticflickr.com/910/28197195808_eee9cb4f5b_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 6" src="https://farm1.staticflickr.com/910/28197195808_aa7744fb80_c.jpg"></a>
村の広場。商店、米線屋などがある。バイクの修理屋さんもあった。

<br>

<a href="https://farm1.staticflickr.com/830/40262166680_573e9c87a4_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 7" src="https://farm1.staticflickr.com/830/40262166680_1c18353ebe_c.jpg"></a>
村の中は坂や階段が多い。

<br>

<a href="https://farm1.staticflickr.com/826/28196666818_1f2d94411a_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 8" src="https://farm1.staticflickr.com/826/28196666818_6dc7188e9f_c.jpg"></a>
古い民家。

<br>

<a href="https://farm1.staticflickr.com/909/27199389617_c69dc97f9f_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 9" src="https://farm1.staticflickr.com/909/27199389617_4a058877a4_c.jpg"></a>

<br>


<a href="https://farm1.staticflickr.com/824/42025247252_a3dc5e4c11_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 10" src="https://farm1.staticflickr.com/824/42025247252_9a0a926b23_c.jpg"></a>

<a href="https://farm1.staticflickr.com/829/41348873724_a75f688ed1_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 11" src="https://farm1.staticflickr.com/829/41348873724_0731514d9e_c.jpg"></a>

<a href="https://farm1.staticflickr.com/826/28197155168_aa61f0c38f_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 12" src="https://farm1.staticflickr.com/826/28197155168_10152730a2_c.jpg"></a>

<a href="https://farm1.staticflickr.com/975/28196626898_bb7a77c9ea_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 13" src="https://farm1.staticflickr.com/975/28196626898_c299194cfa_c.jpg"></a>

<br>

<a href="https://farm1.staticflickr.com/958/40261163540_6e04384c1e_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 14" src="https://farm1.staticflickr.com/958/40261163540_9ff78b32dc_c.jpg"></a>

<br>

<a href="https://farm1.staticflickr.com/972/42069593841_b65b32780a_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 15" src="https://farm1.staticflickr.com/972/42069593841_d964574e42_c.jpg"></a>

<br>

<a href="https://farm1.staticflickr.com/948/42025354412_66feb9e5bc_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 16" src="https://farm1.staticflickr.com/948/42025354412_5f82f2b16d_c.jpg"></a>
板板橋村からさらに進んで蒋家坪村まで行く。

<br>

<a href="https://farm1.staticflickr.com/974/41348861414_cbf6e4f290_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 17" src="https://farm1.staticflickr.com/974/41348861414_4bdd79070a_c.jpg"></a>
15分ほど歩いて蒋家坪に着く。この村はヤオ族とミャオ族が雑居している。雑居といっても村の人に聞くと上の方にヤオ族、下の方にミャオ族と一応の住み分けはあるようだ。

<br>

<a href="https://farm1.staticflickr.com/907/41348945564_6bdc36f813_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 18" src="https://farm1.staticflickr.com/907/41348945564_ca390b84b7_c.jpg"></a>

<br>

<a href="https://farm1.staticflickr.com/951/41169863515_264ba3dd06_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 19" src="https://farm1.staticflickr.com/951/41169863515_76360ef930_c.jpg"></a>

<a href="https://farm1.staticflickr.com/946/42069146391_c51396d810_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 25" src="https://farm1.staticflickr.com/946/42069146391_f85555ce11_c.jpg"></a>

<br>

<a href="https://farm1.staticflickr.com/962/40261898550_d21a2bf9ab_o.jpg"><img alt="ヤオ族とミャオ族の村を見に金平から板板橋と蒋家坪までトレッキング, 26" src="https://farm1.staticflickr.com/962/40261898550_114325b66c_c.jpg"></a>
