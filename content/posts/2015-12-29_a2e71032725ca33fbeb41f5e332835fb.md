 <a href="https://farm1.staticflickr.com/969/41349813974_86ed6b05ca_o.jpg"><img alt="金平からハニ族の老寨までトレッキング" src="https://farm1.staticflickr.com/969/41349813974_5b6b6663a5_c.jpg"></a>
今日は鼠の日で金平のマーケットを見る予定だったのだが、2014年から金平が日曜マーケットになってしまっていたので、今日はやることない。仕方ないので今日もぶらぶらと歩きに行く。11時半、いつものように広場から山を登って行く。

<br>

<a href="https://farm1.staticflickr.com/972/28197023888_72368636c4_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 1" src="https://farm1.staticflickr.com/972/28197023888_6f3aac332b_c.jpg"></a>
45分ほどでハニ族の村に出る。

<br>

<a href="https://farm1.staticflickr.com/827/28197078568_8fc2632cdb_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 2" src="https://farm1.staticflickr.com/827/28197078568_36f6866485_c.jpg"></a>
ハニ族の村から30分で板板橋村との分岐に着く。今日は板板橋村には行かないので左に進む。

<br>

<a href="https://farm1.staticflickr.com/946/28196603048_fa3fd132dd_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 3" src="https://farm1.staticflickr.com/946/28196603048_2e9f6741a8_c.jpg"></a>
湖があり車でピクニックに来ている人や釣りをしてる人がいる。

<br>

<a href="https://farm1.staticflickr.com/911/42068930861_7f2c6892a1_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 4" src="https://farm1.staticflickr.com/911/42068930861_ef5928cb79_c.jpg"></a>
道の脇に茶碗がよく置かれているがこれは何かの供物なのかな。供物にしてはちょっとこれを供されてもという気がする。

<br>

<a href="https://farm1.staticflickr.com/903/28196117678_cefc3ec452_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 5" src="https://farm1.staticflickr.com/903/28196117678_3510b22c12_c.jpg"></a>
分岐から20分ほどでハニ族の村・老寨に着く。

<br>

<a href="https://farm1.staticflickr.com/981/41168600275_55c1bf093f_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 6" src="https://farm1.staticflickr.com/981/41168600275_6c29f78e4c_c.jpg"></a>
養蜂箱が壁にかかっている家が多い。こうやっていると勝手に蜜が集まるのだな、なかなかいいものだと思ったが、きっとぼくが考えているよりずっと手のかかるものなのだろう。甘味とは貴重なものでそう簡単には手に入らないのだ。

<br>

<a href="https://farm1.staticflickr.com/960/42024956172_fcd94b5f6a_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 7" src="https://farm1.staticflickr.com/960/42024956172_a32cf31304_c.jpg"></a>
洗濯を干していたり。

<br>

<a href="https://farm1.staticflickr.com/950/28197302768_64c1ce65ae_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 8" src="https://farm1.staticflickr.com/950/28197302768_4ed28de654_c.jpg"></a>
何かを脱穀していたり。みなさん働いている。勝手に蜜が集まるだのと楽なことばかり思っていたキリギリス的自分が恥ずかしい。

<br>

<a href="https://farm1.staticflickr.com/912/41348851384_0914fdcfc7_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 9" src="https://farm1.staticflickr.com/912/41348851384_8eab487aee_c.jpg"></a>
ブランコがあった。

<br>

<a href="https://farm1.staticflickr.com/971/42069529261_fdda3c1c75_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 10" src="https://farm1.staticflickr.com/971/42069529261_40e0b2453f_c.jpg"></a>
村を過ぎると緩やかな下りになる。景色がいい。

<br>

<a href="https://farm1.staticflickr.com/912/40262152840_24854a4b41_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 11" src="https://farm1.staticflickr.com/912/40262152840_1e14b7b298_c.jpg"></a>
道なりに行くと金平からだいぶ南の方に下りて行きそうなので山道に入って山を下る。

<br>

<a href="https://farm1.staticflickr.com/952/28197328408_60d2b35cc6_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 12" src="https://farm1.staticflickr.com/952/28197328408_ed1ec79cfd_c.jpg"></a>
山道は畑の中を通りながら下っていく。

<br>

<a href="https://farm1.staticflickr.com/973/41348835874_c5fdd81149_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 13" src="https://farm1.staticflickr.com/973/41348835874_5792066a92_c.jpg"></a>
しかし畑の中で道がなくなる。また畑のあぜ道を歩いて下っていくとハニ族のおじさんがいたのでどうやって下まで下りればいいか聞くと、少し大きなあぜ道まで案内してくれた。あとは下にある小さな橋を渡れば金平に行く道にでるらしい。

<br>

<a href="https://farm1.staticflickr.com/957/27198321437_1136925c0f_o.jpg"><img alt="金平からハニ族の老寨までトレッキング, 14" src="https://farm1.staticflickr.com/957/27198321437_c0fc949f78_c.jpg"></a>
橋を渡り少し歩くと車道に出た。あとは車道を歩いて金平に戻る。3時半、金平に着く。

<br>
