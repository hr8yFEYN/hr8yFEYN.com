 <a href="https://farm1.staticflickr.com/966/42024080882_a2b23b9fed_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2" src="https://farm1.staticflickr.com/966/42024080882_1e97290281_c.jpg"></a>
公安のお墨付きをもらいニャーロン(新龍)まで歩けることになった。公安の気が変わらないうちにとっとと出発する。

<br>

<a href="https://farm1.staticflickr.com/905/42024437762_3b7d3fda7f_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 1" src="https://farm1.staticflickr.com/905/42024437762_2599133cd7_c.jpg"></a>
シャーバ(下壩、下坝)から川沿いを北にしばらく行くとゴンパがあったのでもう公安に捕まらないように仏様を拝んでおこうと寄るが閉まっていて中に入れない。

<br>


<a href="https://farm1.staticflickr.com/974/40260988110_ec518c923b_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 2" src="https://farm1.staticflickr.com/974/40260988110_2fbb02e3fa_c.jpg"></a>
天気がよく川沿いを気持ちよく歩いていると雨が降ってきたのでテントを張る。

<br>

<a href="https://farm1.staticflickr.com/977/42069573011_2d30a1acc6_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 3" src="https://farm1.staticflickr.com/977/42069573011_b3eb57bd89_c.jpg"></a>
5日目。歩き始めるとすぐにモーバ(莫坝)に着く。川沿いに集落が広がっている。

<br>

<a href="https://farm1.staticflickr.com/956/40261974410_c80cd45fc0_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 4" src="https://farm1.staticflickr.com/956/40261974410_b7c1e78230_c.jpg"></a>
川沿いを離れ山の中腹を進んで行くと開けた場所に出る。ヤクが放牧されている。

3時半、雨が降ってきたので歩くのをやめる。

<br>

<a href="https://farm1.staticflickr.com/971/41169551135_877f8aba7b_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 5" src="https://farm1.staticflickr.com/971/41169551135_8f0ab03936_c.jpg"></a>
6日目。歩き始めるとすぐに人の住んでいなさそうな集落がある。共産党が遊牧民を定住させるために作ったものなのかな。

<br>

<a href="https://farm1.staticflickr.com/969/40262108370_6e0bb0a793_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 6" src="https://farm1.staticflickr.com/969/40262108370_c559b90830_c.jpg"></a>
緩やかに登って行き峠を越える。

<br>

<a href="https://farm1.staticflickr.com/828/28196642258_fc6820b7c2_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 7" src="https://farm1.staticflickr.com/828/28196642258_76cf653d92_c.jpg"></a>
峠を越えると急な下り。調子に乗って下っていたら膝を捻ってしまった。それほど痛みもないのでそのまま下る。

<br>

<a href="https://farm1.staticflickr.com/975/41169250165_eefa5cf71f_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 8" src="https://farm1.staticflickr.com/975/41169250165_ff8d6cc9e8_c.jpg"></a>
7日目、小雨。昨日、捻った膝が少し痛む。1時間ほど下ってピッツァ(皮察)に着く。商店で買い物をする。

<br>

<a href="https://farm1.staticflickr.com/959/41168563105_bd74da3172_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 9" src="https://farm1.staticflickr.com/959/41168563105_df6943ed83_c.jpg"></a>
深い谷間の道を下って行く。ぼこぼことはいえリタン以来の舗装路。天気がよくなる。

<br>

<a href="https://farm1.staticflickr.com/949/42024985832_d91e27ff8e_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 10" src="https://farm1.staticflickr.com/949/42024985832_81ffee1c1e_c.jpg"></a>
2時半、省道217線に出る。立派な舗装路。膝が痛むが我慢して歩く。

<br>

<a href="https://farm1.staticflickr.com/981/41348921004_0e5f454ee3_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 2, 11" src="https://farm1.staticflickr.com/981/41348921004_8e70b5139e_c.jpg"></a>
8日目。膝がだいぶ痛むが新龍までもう少しなので歩くことにする。昼前、新龍に着く。

新龍(ニャーロン)で1週間ほど膝の様子をみてみたがいっこうによくならないので、徒歩旅行はここで敗退することにする。来年、膝がよくなったらまたここから歩き始めようと思う。
