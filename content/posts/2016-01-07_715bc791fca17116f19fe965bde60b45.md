 昨日、者米からの老勐行きバスの時間を聞いたら道の状況が悪く現在は運行されていなといわれた。明日、老勐では定期市が開かれるので明日の午前中できるだけ早い時間に着きたいのだ。者米からバスで老勐に行くには金平に戻り、翌日の朝、老勐行きのバスに乗る。しかしこれだと老勐に着くのは早くても昼頃になってしまう。困った。金平でバスが運行されているか確認しておくべきだった。

地図で見ると者米から老勐は山をひとつ越えれば行ける。山の上に老集寨という町があり老集寨からは金平と元陽行きのバスが出ている。どちらのバスも老勐を通る。老集寨から老勐まではバスで1時間程度なので明日の朝のバスに乗れば十分にマーケットに間に合う。GPSで直線距離を調べると者米から老集寨までは12.5キロ。山の道なので倍の距離があるとしても25キロ。老集寨までずっと登りでも10時間も歩けば着けるだろう。東チベットの理塘から新龍を歩いている時に膝を捻挫してから膝を大事にしてきたが久しぶりに歩いてやるぞ。覚悟しておけよ、ぼくの膝。

<a href="https://farm1.staticflickr.com/965/41350081264_5f9566ec46_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 2" src="https://farm1.staticflickr.com/965/41350081264_a05953deb6_c.jpg"></a>

橋を渡る。

<a href="https://farm1.staticflickr.com/981/28196717838_9b8e0379cb_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 4" src="https://farm1.staticflickr.com/981/28196717838_3239b3bff3_c.jpg"></a>

<a href="https://farm1.staticflickr.com/956/42068826091_edf88bcaf4_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 6" src="https://farm1.staticflickr.com/956/42068826091_198d80575e_c.jpg"></a>
ここから登り。急な登りで道もぬかるんでいて歩きにくい。

<br>

<a href="https://farm1.staticflickr.com/943/42069202081_15522084df_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 7" src="https://farm1.staticflickr.com/943/42069202081_74db7790dd_c.jpg"></a>
だいぶ登ってきた。者米から勐拉に向かう道がだいぶ下の方に見える。

<br>

<a href="https://farm1.staticflickr.com/825/28196806298_6b6d71ffee_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 8" src="https://farm1.staticflickr.com/825/28196806298_289d46e73e_c.jpg"></a>
10時、村に着く。ここはアールー族の村のようだ。者米から老孟まではやはりアールー族が多く住んでいるようだ。村の人に老集寨に行く道を聞いて歩き始める。村で少しのんびりとしたいが老集寨までどれくらい時間がかかるかわからないので急ぐ。

<br>

<a href="https://farm1.staticflickr.com/981/41349683504_768a503844_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 9" src="https://farm1.staticflickr.com/981/41349683504_1096ae18af_c.jpg"></a>
村を過ぎると道は緩やかな登りになる。霧が濃い。

<br>

<a href="https://farm1.staticflickr.com/905/27198868087_da83b70197_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 10" src="https://farm1.staticflickr.com/905/27198868087_1f6d1acc96_c.jpg"></a>
ひたすら歩く。アールー族がバイクに乗って通り過ぎたり、畑で作業をしている。

<br>

<a href="https://farm1.staticflickr.com/973/41168976625_8503b6d488_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 11" src="https://farm1.staticflickr.com/973/41168976625_36926da610_c.jpg"></a>
14時、また村に着く。第一村人を発見。老集寨までの道を聞く。村を上に登って行けとのこと。

<br>

<a href="https://farm1.staticflickr.com/960/42024705522_532b3bd51f_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 12" src="https://farm1.staticflickr.com/960/42024705522_2c9a44732d_c.jpg"></a>
第二村人を発見。老集寨までの道を聞く。村を上に登って行けとのこと。登っていくと舗装された車道に出た。この道が老集寨に行くそうだ。しばらく歩くと羅盤という村で道が分かれる。老集寨へは右に行く。この辺りで峠を越えたようで道は下りになる。

<br>

<a href="https://farm1.staticflickr.com/972/28197039208_57df8c8c85_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 13" src="https://farm1.staticflickr.com/972/28197039208_ff63f7eb35_c.jpg"></a>
ぐちょぐちょというか水没してしまった道。歩き難い。

<br>

<a href="https://farm1.staticflickr.com/830/27199797937_158b34fb2b_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 14" src="https://farm1.staticflickr.com/830/27199797937_617b9e7952_c.jpg"></a>
15時、ハニ族の村に着く。

<br>

<a href="https://farm1.staticflickr.com/964/42068516851_105029c52c_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 15" src="https://farm1.staticflickr.com/964/42068516851_6936b7e446_c.jpg"></a>
15時半、ハニ族の村から舗装路を30分歩くと、老集寨に着いた。10時間ぐらいかかるとみていたが7時間半で来れた。思っていたより近かった気がするが、GPSを見ると24キロだったので予測していたような距離だった。

<br>

<a href="https://farm1.staticflickr.com/951/42068766201_20bd4a2c6d_o.jpg"><img alt="者米から歩いて山を越え老集寨へ, 16" src="https://farm1.staticflickr.com/951/42068766201_2318defa00_c.jpg"></a>
老集寨には宿がひとつしかなく50元でトイレシャワー共同、WiFiはもちろんなしととても高いが他に選択はない。40元にまけてもらう。金平と元陽行きのバスが7:00と7:20に出るので老勐の日曜市を見逃さずに済みそうだ。

者米から老集寨の道は天気が良ければもっと楽しそうだ。者米から老集寨は登りだが、老集寨から者米までは当然下りなので老集寨・者米のトレッキングの方がいいかもしれない。アールー族の村をいくつも通るアールー族街道なのでおすすめ。
