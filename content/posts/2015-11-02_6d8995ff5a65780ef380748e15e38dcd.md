ビザの更新などで1ヶ月ほど時間が空いてしまったがリタン(理塘)からまた徒歩旅行を始める。とりあえずはニャーロン(新龍)を目指す。

リタンからニャーロンまでは省道217線が通っているが車通りの多い舗装路を歩いてもつまらなそうなので州道217線の西を走る郷道004線を歩くことにする。

<br>

<a href="https://farm1.staticflickr.com/909/42024884692_2fd419ff9d_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 1" src="https://farm1.staticflickr.com/909/42024884692_bba0391ee3_c.jpg"></a>
1日目。8時半、歩き始める。リタン市内から郷道004線を歩いて山を登って行くよりもリタン・ゴンパから沢づたいに山を登ってから郷道004線に合流する方が距離が短そうなのでリタン・ゴンパから沢を登って行くことにする。

<br>

<a href="https://farm1.staticflickr.com/826/42069083181_2b292e2cf4_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 2" src="https://farm1.staticflickr.com/826/42069083181_83621d839b_c.jpg"></a>
2時間半登って郷道004線に合流する。曇ってきて少し肌寒い。

<br>

<a href="https://farm1.staticflickr.com/979/42024530652_f1f139aeb5_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 3" src="https://farm1.staticflickr.com/979/42024530652_7aba90e9e1_c.jpg"></a>
昼にツァンパを食べて、しばらく休憩。

<br>

<a href="https://farm1.staticflickr.com/967/41169032605_77eeb96aa5_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 4" src="https://farm1.staticflickr.com/967/41169032605_ee3df48bfd_c.jpg"></a>
道がつづら折りに下って行くので山の中を歩いて近道をする。

<br>

<a href="https://farm1.staticflickr.com/951/42023772742_7a1883687e_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 5" src="https://farm1.staticflickr.com/951/42023772742_e94ea5eebf_c.jpg"></a>
下りきった辺りで道に戻る。道は川沿いを登って行く。3時頃、橋を渡る。疲れたので歩くのをやめる。

<br>

<a href="https://farm1.staticflickr.com/948/27199774497_81cd662857_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 6" src="https://farm1.staticflickr.com/948/27199774497_fb05ddb839_c.jpg"></a>
2日目。今にも雨が降り出しそうな天気。8時前に出発。歩き始めるとすぐに雨になる。道は登ったり下ったりしながら緩やかに登って行く。

<br>

<a href="https://farm1.staticflickr.com/959/28197234718_5e4777a093_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 7" src="https://farm1.staticflickr.com/959/28197234718_b5a8770556_c.jpg"></a>
1ヶ月のブランクで体力がだいぶ落ちていて緩やかな登りでも休み休みでないと歩けない。3時、歩くのをやめる。だいぶ疲れた。

<br>

<a href="https://farm1.staticflickr.com/974/28196614168_df4e9f5179_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 8" src="https://farm1.staticflickr.com/974/28196614168_2170a659e5_c.jpg"></a>
3日目。8時半に歩き始める。1時間もすると下りになる。川沿いを下って行く。

<br>

<a href="https://farm1.staticflickr.com/909/42068719041_d8f3389a33_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 9" src="https://farm1.staticflickr.com/909/42068719041_6530b4d39a_c.jpg"></a>
だいぶ谷が深くなる。標高が低くなったからか山に松や杉が生えている。

3時になったので歩くのをやめる。谷が深く平らな場所がないので急斜面になんとかテントを張る。

<br>

<a href="https://farm1.staticflickr.com/825/41349783894_47cfabea14_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 10" src="https://farm1.staticflickr.com/825/41349783894_27b286e7b0_c.jpg"></a>
4日目、久しぶりに雨が降らなそうな天気。いい気分で歩いているとシャーバ(下壩、下坝)という村の手前で公安のジープがやって来てこれ以上進んではいけないといわれる、リタンに引き返せと。

どうにも仕方がないので、公安にあなた方の名前となぜこれ以上進んではいけないのかを書いて下さい、成都に戻ったときに四川省の公安局であなた方の措置が中国の法律上正しいことなのか確認しますというと、それはできないという。これはしめたと思い、名前も教えてくれない人のいうことは聞けないし信用もできませんといい無視して進む。公安も諦めたのかシャーバに戻って行く。

<br>

<a href="https://farm1.staticflickr.com/973/40261604740_1636921ae7_o.jpg"><img alt="郷道004線を歩いてリタン(理塘)からニャーロン(新龍)に行く 1, 11" src="https://farm1.staticflickr.com/973/40261604740_86654035b9_c.jpg"></a>
シャーバの商店で買い物をしているとまた公安が来て、商店に併設された食堂に連れて行かれる。四川省の公安に確認をしてみるといって自分の身分証を見せた。

公安は電話をかけたりスマートフォンをいじったりいろいろしている。商店のおじさんが心配そうに見ている。どうしたのと聞かれたので外国人はこれ以上進んではいけないらしいというと、彼は昆明からずっと歩いて来ているんだよ、行かせてあげなよと公安にいってくれる。

30分くらいして確認が終わり、すみませんでした、ここから山を越えて新龍(ニャーロン)に行っても構いませんといわれた。商店のおじさんもすごく嬉しそうな顔をして、いってもいいってよ、よかったねといってくれた。なんとか公安のお墨付きをもらって郷道004線を進んで行けることになった。

しかし実際にシャーバの公安が四川省の公安に確認していたのかは怪しいのではないかと思っている。ぼくが本当に四川省の公安に行って、彼らの措置が正しいのか確認されて面倒なことになるのを嫌って行かせてしまえということになったのではないかと思っている。

その後、この辺りを歩いていて聞いたのだが、以前この辺りで人さらいが何件かあったらしい。中国でよく聞く人をさらって臓器を売ってしまうというやつ。眉唾で聞いていたけど実際にあるのだ。

そういうわけでこの辺りをよそ者が歩いていると、公安といわず村人にもお前はどこから来たのだとか、どこに行くのだとか聞かれた。身分証を見せろとも何度かいわれたが、そういう事情なら仕方ないのかと思う。しかし子供もさらわれたというので全くひどい話だ。
