 <a href="https://farm1.staticflickr.com/972/42023740892_0585f7cd20_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹)" src="https://farm1.staticflickr.com/972/42023740892_bebd92ab3e_c.jpg"></a>
1日目。7時過ぎに宿をチェックアウト。近くの食堂で包子を食べ、歩き始める。 
すぐに雨が降り出しそうな天気。 

<br>

<a href="https://farm1.staticflickr.com/905/41168703075_02e4854e0e_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 1" src="https://farm1.staticflickr.com/905/41168703075_480985c51d_c.jpg"></a>
町を出てしばらく歩くとゆるやかに峠を登って行く。 

<br>

<a href="https://farm1.staticflickr.com/904/40262280860_651dec24f7_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 2" src="https://farm1.staticflickr.com/904/40262280860_365bc30e93_c.jpg"></a>
昼過ぎ町に着く。町を過ぎるとセルシュゴンパが見えてくる。 

<br>

<a href="https://farm1.staticflickr.com/960/42025310972_cdbbaa5783_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 3" src="https://farm1.staticflickr.com/960/42025310972_180ed6d31d_c.jpg"></a>
近くに見えたがセルシュゴンパまで2時間以上かかった。夕方までセルシュゴンパを見学して少し歩いたところでテント泊。 

<br>

<a href="https://farm1.staticflickr.com/971/42024731172_832dfea227_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 4" src="https://farm1.staticflickr.com/971/42024731172_c4d0dbbbbc_c.jpg"></a>
セルシュゴンパから老犬がついてきた。テントを張って食事をするときも近くで伏せている。 
ソーセージを放るがもう鼻が効かなくなっているのかどこに落ちたのかわからないようなので、
目の前に放ると何とか見つけて食べていた。

<br>

<a href="https://farm1.staticflickr.com/951/41349696794_b0dae99cce_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 5" src="https://farm1.staticflickr.com/951/41349696794_ae0bda0d2c_c.jpg"></a>
2日目。朝起きると霜がバリバリに下りていた。犬はいなくなっていた。 

歩き始めて1時間もすると町に着く。道路にブレーキを踏んだ跡があり、 
その先に犬の死骸があった。 昨日の犬ではないといいなと思った。

<br>

<a href="https://farm1.staticflickr.com/945/42024304652_7539040e65_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 6" src="https://farm1.staticflickr.com/945/42024304652_5a3818f437_c.jpg"></a>
ぽつりぽつりと集落がある。集落の家は新しいセメントの家。遊牧もしていないようだし、農業も当然していないようだ。 
どうやって生計を立てているいるのだろう。小さな集落にも学校や政府の庁舎などが造られていたし、道路や橋も多く造っているので、 
そういう仕事があるのかもしれない。 

夕方まで歩きテント泊。 

<br>

<a href="https://farm1.staticflickr.com/970/42024802152_e3d4eeea8a_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 7" src="https://farm1.staticflickr.com/970/42024802152_31f5050bc6_c.jpg"></a>
3日目。歩き始めるとすぐにゆるやかな峠を登り始める。 

<br>

<a href="https://farm1.staticflickr.com/910/42024301082_6f768ff613_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 8" src="https://farm1.staticflickr.com/910/42024301082_1ac836c4fb_c.jpg"></a>
峠が四川省と青海省の省境。 

<br>

<a href="https://farm1.staticflickr.com/946/40261576800_f7ed9eb8b7_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 9" src="https://farm1.staticflickr.com/946/40261576800_be90029175_c.jpg"></a>
旧道を下る。だいぶ近道。 峠を下り車道と合流する。山間が狭くなる。 

<br>

<a href="https://farm1.staticflickr.com/826/42024654882_c0de110e54_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 10" src="https://farm1.staticflickr.com/826/42024654882_a16a8a5aa0_c.jpg"></a>
昼前に町に着く。食堂で焼きそばを食べる。 

<br>

<a href="https://farm1.staticflickr.com/948/41169562305_224d9da36e_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 11" src="https://farm1.staticflickr.com/948/41169562305_a14c5f909f_c.jpg"></a>
ゴンパを見学。お堂も僧坊も新しいのでこの辺りの建物もジェクンドの地震の時に倒壊したのかもしれない。町の建物も新しい。 

<br>

<a href="https://farm1.staticflickr.com/912/40261584580_da6e9cd6c1_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 12" src="https://farm1.staticflickr.com/912/40261584580_6418560def_c.jpg"></a>
ここからは国道の立派な道。となりには高速も走っている。 

<br>

<a href="https://farm1.staticflickr.com/824/41348704954_148ef734b0_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 13" src="https://farm1.staticflickr.com/824/41348704954_fb2a2044aa_c.jpg"></a>
夕方、豪雨。しばらく雨宿りして、テントを張れそうな場所を見つけテント泊。

<br>

<a href="https://farm1.staticflickr.com/947/28196289588_f54c6297e3_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 14" src="https://farm1.staticflickr.com/947/28196289588_dc18a2d15e_c.jpg"></a>
4日目。小雨が降っている。

<br>

<a href="https://farm1.staticflickr.com/956/28196445878_ed0431beca_o.jpg"><img alt="セルシュ(石渠)からジェクンド(玉樹), 15" src="https://farm1.staticflickr.com/956/28196445878_6d5082379d_c.jpg"></a>
国道をだらだらと歩き、3時にジェクンドに着く。

今回、出発するときに何となくジェクンドまで歩けたらいいかなと思っていたが、
思っていたよりずっと早くジェクンドに着いてしまった。もう少し先まで歩こうかとほんの少しだけ思ったが、
気分はもうカンゼの温泉で湯治ですもんね。

<br>
