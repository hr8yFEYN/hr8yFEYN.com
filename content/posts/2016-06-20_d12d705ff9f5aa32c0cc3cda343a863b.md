ガンゼ自治州白玉県の県城。チベット自治省まで12キロ。大きな町でスーパーやファストフード店もある。

# アクセス・行き方

## 康定
{% travel_info_item(img="https://farm1.staticflickr.com/957/42069928011_0d468a6cfa_c.jpg", link="https://farm1.staticflickr.com/957/42069928011_3e71046a9e_o.jpg", alt="康定") %}

康定バスターミナルから白玉(ペユル)行きのバスが出ている。朝6時発、所要14時間、200元。

市内から河東街を南に1キロ程度行ったところに白玉バスターミナルがあり、6:00に康定行きのバスが出ている。所要14時間、203元。

{% end %}


## カンゼ
漢藏大酒店の前辺りでミニバンが客待ちをしている。80〜100元(この値段は言い値なので値切れるかもしれない)。

## デルゲ(徳格)
漢藏大酒店の前辺りでミニバンが客待ちをしている。70〜80元(この値段も言い値なので値切れるかもしれない)。

# 宿泊
白玉バスターミナル前の道、河東街を歩いて市内に入って来ると公安局の手前の集合住宅に宿が多く入っている。トイレシャワー付きの部屋で100元程度、トイレシャワー共同だと70〜80元。多少は値切れる。登記はあまりうるさくないようで外国人も泊めてくれる。
 どの宿もWiFiを備えているが、遅かったり繋がらない場合もあるのでチェックイン前に確認した方がいい。

## 漢藏大酒店
{% travel_info_item(img="https://farm1.staticflickr.com/905/42025240232_543b9387cd_c.jpg", link="https://farm1.staticflickr.com/905/42025240232_cebb8ca8e8_o.jpg", alt="漢藏大酒店") %}

市内にある中では一番大きなホテル。ツイン 198元。

{% end %}


## 鵬東旅館
{% travel_info_item(img="https://farm1.staticflickr.com/832/42068732601_143b486eb4_c.jpg", link="https://farm1.staticflickr.com/832/42068732601_705635f86e_o.jpg", alt="鵬東旅館") %}

バスターミナルから河東街を市内に向かい公安の手前を左に入ったところにある。トイレシャワー付きの部屋 100元〜。トイレシャワー共同の部屋 80元。ドミトリー 30元。

{% end %}


# 観光

## ペユル・ゴンパ
{% travel_info_item(img="https://farm1.staticflickr.com/977/40262279400_c271331d38_c.jpg", link="https://farm1.staticflickr.com/977/40262279400_7a11544d04_o.jpg", alt="ペユル・ゴンパ") %}

ニンマ派六大僧院のひとつ。市内にあるので歩いていける。

{% end %}


## カトク・ゴンパ
{% travel_info_item(img="https://farm1.staticflickr.com/830/42024560882_528417e83b_c.jpg", link="https://farm1.staticflickr.com/830/42024560882_894b9f6b8e_o.jpg", alt="カトク・ゴンパ") %}

こちらもニンマ派六大僧院のひとつ。中国語では噶蛇寺または噶拖寺。公共の交通機関はないので、ミニバンをチャーターして行く。言い値は往復で300元。

ゴンパに商店、食堂、宿泊施設(1ベッド 50元)あり。

{% end %}



## ゾンサル・ゴンパ
{% travel_info_item(img="https://farm1.staticflickr.com/952/41170046475_f436813318_c.jpg", link="https://farm1.staticflickr.com/952/41170046475_0fd90fd7fe_o.jpg", alt="ゾンサル・ゴンパ") %}

サキャ派のお寺。中国語では宗薩寺。ミニバンをチャーターして行く。言い値は往復で400元。

ゴンパの周りに商店、食堂、宿など多数あり。宿はドミトリーで20〜30元。殆どの宿にシャワーはない。WiFiなどインターネット環境もない。

{% end %}


## ペルプン・ゴンパ
{% travel_info_item(img="https://farm1.staticflickr.com/944/40262559780_7b947312ab_c.jpg", link="https://farm1.staticflickr.com/944/40262559780_18d67fc2b6_o.jpg", alt="ペルプン・ゴンパ") %}

カルマ・カギュ派の三大僧院のひとつ。中国語では八邦寺。ペユルでミニバンをチャーターできる。言い値は往復で400元。

ゾンサルゴンパとペルプンゴンパ間には山道があり歩いて行くこともできる。朝ゾンサルゴンパを出ると夕方にはペルプンゴンパに着く。4,500メートルの峠を越える。

ゴンパ下の八邦という町に商店、食堂あり。和平旅館という宿がある(ドミトリー 30元)。

{% end %}
