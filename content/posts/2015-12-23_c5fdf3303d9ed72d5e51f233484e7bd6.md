前日に宿のおじさんにバスの時間を聞いたら、7時、8時、9時半頃だというので、7時前に宿を出る。宿を出るとバスがもう停まっていた。

バスは7時20分に発車。金平までは40元(800円)。

45分ほど走り、蛮耗の町ではなく高速の入り口で朝食休憩。高速に乗らないのになぜわざわざ高速の入り口で休憩なのか。きっとここの食堂と癒着しているのだ。

30分ほど休憩をして出発するが、今度は蛮耗の町で客待ち。

<br>

<a href="https://farm1.staticflickr.com/977/41349953494_931fb2024e_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 1" src="https://farm1.staticflickr.com/977/41349953494_2fa4b85e07_c.jpg"></a>
蛮耗から金平は新しい道ができていて1時間ちょっとで金平に着く。新しいバスターミナルもできていてそこに停車。10年ぐらい前にも新しいバスターミナルを造っていたが新バスターミナルも10年で旧バスターミナルになってしまったのか。

新バスターミナルは北バスターミナルという名前なので、以前の新バスターミナルは南バスターミナルということになっているのかもしれない。

<br>

<a href="https://farm1.staticflickr.com/958/41349457854_4931c18712_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 2" src="https://farm1.staticflickr.com/958/41349457854_7f7c1decc7_c.jpg"></a>
金平に来たらいつも泊まる世昌旅館に行く。まだ営業してるか不安だったがまだ営業していた。トイレ・シャワー共同のシングルが30元(600円)。以前は20元だったので10元の値上げ。WiFiなしで設備もだいぶ老朽化している。

<br>

<a href="https://farm1.staticflickr.com/955/42068509041_fa33966967_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 3" src="https://farm1.staticflickr.com/955/42068509041_9990d4886e_c.jpg"></a>
宿も決まったので町歩きに行く。とりあえず広場に行ってみる。広場には工事中。囲いができるようだ。

<br>

<a href="https://farm1.staticflickr.com/958/41169581205_5e27f55503_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 4" src="https://farm1.staticflickr.com/958/41169581205_ae285542cc_c.jpg"></a>
広場の周りには日雇い仕事を求める人がたくさんいる。11時近くなので今日はもう仕事はなさそうだ。

<br>

<a href="https://farm1.staticflickr.com/956/41169267865_4976dc4ef5_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 5" src="https://farm1.staticflickr.com/956/41169267865_27b8d84339_c.jpg"></a>
以前によく食べていた面条屋(ラーメン屋)に行ってみるが大きなマンションになっていた。近くに米線屋はたくさんあるが面条を食べたいので、昔のバスターミナルを見に行きながら面条屋を探す。

しばらく歩くと面条を扱う店があったので面条を食べる(6元、120円)。米線よりも美味しいというわけではないが米線ばかり食べていたので米線よりも美味しく感じる。

<br>

<a href="https://farm1.staticflickr.com/973/28196578958_eb487d0f28_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 6" src="https://farm1.staticflickr.com/973/28196578958_316b6c5186_c.jpg"></a>
南バスターミナルからは勐拉と那发に行くミニバンが出ていた。どちらも所要1時間程度で20元とのこと。

<br>

<a href="https://farm1.staticflickr.com/966/41349192794_fb126294ac_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 7" src="https://farm1.staticflickr.com/966/41349192794_cedea7ca55_c.jpg"></a>
市場をのぞいてみる。市場はあまり以前と変わっていない。

<br>

<a href="https://farm1.staticflickr.com/906/41350184064_0d0bea27be_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 8" src="https://farm1.staticflickr.com/906/41350184064_ea28c4d682_c.jpg"></a>
よくきな粉もちを食べていたところでまだきな粉もちを売っていたので2元分注文する。

<br>

<a href="https://farm1.staticflickr.com/957/41349018934_955d09bafc_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 9" src="https://farm1.staticflickr.com/957/41349018934_13a67bbded_c.jpg"></a>
きな粉もちが出てくると思っていたが甘い蜜を掛けたものもが出てきた。すっかりきな粉もちが出てくると思い込んでしまっていたが、ちゃんときな粉味といわないといけないようだ。また明日食べに来なくては。

<br>

<a href="https://farm1.staticflickr.com/964/42024659252_0ac075717a_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 10" src="https://farm1.staticflickr.com/964/42024659252_83c5aa032d_c.jpg"></a>
市場というか金平の町では民族衣装を着た人を見かける。ヤオ族、ハニ族、ミャオ族が多い。

市場を抜けると世昌旅館に出るが、もう少し歩いてから宿に戻ることにする。

<br>


<a href="https://farm1.staticflickr.com/823/41169584875_7a9ce72939_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 11" src="https://farm1.staticflickr.com/823/41169584875_4bb7f78079_c.jpg"></a>
金平は坂や階段が多い。

<br>

<a href="https://farm1.staticflickr.com/827/40261451440_4f61aef6cb_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 12" src="https://farm1.staticflickr.com/827/40261451440_046e01882f_c.jpg"></a>
路地に坂や階段があったりして路地を歩くのも楽しい。野菜や焼きもろこしが狭い路地で売られていたりする。

<br>

<a href="https://farm1.staticflickr.com/961/41349467824_98f7fd28e8_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 13" src="https://farm1.staticflickr.com/961/41349467824_0f10437c89_c.jpg"></a>
路地を下り、川を渡ると十里村行きのミニバス乗り場があった。

<br>

<a href="https://farm1.staticflickr.com/959/41350305504_602cd2f3e0_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 14" src="https://farm1.staticflickr.com/959/41350305504_711b48d615_c.jpg"></a>
市内の外れは建設ラッシュ。ほとんど人の住んでいなそうなマンションも多い。

<br>


<a href="https://farm1.staticflickr.com/971/42024629602_52c536506f_o.jpg"><img alt="勐橋(勐桥)から金平に行き、金平をぶらぶら歩き, 15" src="https://farm1.staticflickr.com/971/42024629602_9cf6f6fffa_c.jpg"></a>
ぶらぶら歩いて町の中心に戻る。世昌旅館の裏に大きな建物が建っていて市場になっているで見に行く。勐橋であれだけバナナを作っているのだからバナナは安いのかなと思って値段を聞いてみたが500グラム2元(40円)とそれほど安くない。勐橋の市場だともっと安かったのかもしれない。

疲れたので宿に戻る。

<br>
