# 寧南

<a href="https://farm1.staticflickr.com/948/42069104931_ec4b7b9f8f_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く" src="https://farm1.staticflickr.com/948/42069104931_8e6fbfa25a_c.jpg"></a>
清翠国際青年旅舎(Skylark Lodge)をチェックアウトして歩いてバスターミナルへ行く。
寧南行きのバスは頻発しているそうなのでチケットを買ったらすぐに出発かと思ったら、
ちょうどバスが出たばかりで次のバスまで1時間あるらしい。

<br>

<a href="https://farm1.staticflickr.com/823/28196713648_15dd7b7088_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 1" src="https://farm1.staticflickr.com/823/28196713648_8d3917de81_c.jpg"></a>
西昌を出るとすぐに山に入っていく。途中、洗車と食事休憩。
中国の洗車はホースからジャーと水をかけるだけで、
拭いたりしないので濡れた車体にまたほこりがついてすぐに汚れる。
たぶんきれいにするということではなく、
洗車をしたということが重要なのだ。

<br>

<a href="https://farm1.staticflickr.com/944/40262354350_89ca6e0e7f_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 2" src="https://farm1.staticflickr.com/944/40262354350_29d789f5a5_c.jpg"></a>
14時半、西昌から3時間ほどで寧南に着く。
朝から何も食べてなかったので、バスターミナル近くの宿に部屋を取り、すぐに食事に行く。

<br>

<a href="https://farm1.staticflickr.com/827/27199350937_6754a41707_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 3" src="https://farm1.staticflickr.com/827/27199350937_42fc5b4ebb_c.jpg"></a>
宿から繁華そうな方に向かって歩くが、よさそうな食堂がない。
もうお腹が限界なので適当なところで土鸡面条(地鶏麺)を食べる。
味はいまいちだったが、成都や西昌と比べるとだいぶ安かった。

<br>

<a href="https://farm1.staticflickr.com/948/27198747517_369f3049ca_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 4" src="https://farm1.staticflickr.com/948/27198747517_98876668ca_c.jpg"></a>
食事も済ませたので町歩き。町には民族衣装を着た年配のイ族の女性が多い。
しかしそれよりもお年寄りばかりだ。
四川省の省都・成都から5時間ぐらいだし若い人にはちょうど出て行きやすい距離なのか。

<br>

<a href="https://farm1.staticflickr.com/975/42069142681_cc93d09646_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 5" src="https://farm1.staticflickr.com/975/42069142681_9d2f6cb3d1_c.jpg"></a>
古い民家も少し残っている。西昌の古鎮の建築とは少し違う。

<br>

<a href="https://farm1.staticflickr.com/949/42069237881_db0910e77e_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 6" src="https://farm1.staticflickr.com/949/42069237881_e49e836e69_c.jpg"></a>
お年寄りは、麻雀やって、お茶のんで、タバコのんで。だいたいどこでも同じですな。

<br>

<a href="https://farm1.staticflickr.com/950/41348941354_0c37764ec3_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 7" src="https://farm1.staticflickr.com/950/41348941354_cddd118d7d_c.jpg"></a>
しかしお年寄りしかいない田舎町なのに、町のはずれにマンションをたくさん造っている。
若い人が戻って来たときのためなのか、ただの過剰開発なのか。

<br>

<a href="https://farm1.staticflickr.com/830/27199721047_76aa188bfb_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 8" src="https://farm1.staticflickr.com/830/27199721047_5565f655b6_c.jpg"></a>
こんなに大きなホテルもこさえている。

<br>

<a href="https://farm1.staticflickr.com/906/28195888548_617e50fe10_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 9" src="https://farm1.staticflickr.com/906/28195888548_239f53649e_c.jpg"></a>
道だってこんなに立派だ。

<br>

<a href="https://farm1.staticflickr.com/911/40262156470_5efd9d86b9_o.jpg"><img alt="西昌から寧南、巧家によって昭通に行く, 10" src="https://farm1.staticflickr.com/911/40262156470_abcd842f3b_c.jpg"></a>
無駄に大きなものばかり見て疲れたので涼粉を食べて町歩きを切り上げた。

<br>

# 巧家

<a href="https://farm1.staticflickr.com/948/28197064398_73d7db79a8_o.jpg"><img alt="巧家" src="https://farm1.staticflickr.com/948/28197064398_38fbc5be7d_c.jpg"></a>
朝のバスで寧南から巧家に向かう。巧家は四川省と雲南省の省境の雲南省側にある。
本当は寧南から昭通に行きたかったのだが、バスターミナルで聞くと、
寧南からは昭通行きのバスは出ていないので巧家で乗り換えろとのことだったので、
巧家に行き、2泊してから昭通に行くことにする。

<br>

<a href="https://farm1.staticflickr.com/955/42069988781_fbb3bd66cc_o.jpg"><img alt="巧家, 1" src="https://farm1.staticflickr.com/955/42069988781_d3f05eb5c6_c.jpg"></a>
寧南からまた山道を走り、寧南から2時間半ほどで巧家に着く。

<br>

<a href="https://farm1.staticflickr.com/969/27198470517_1bd215af78_o.jpg"><img alt="巧家, 2" src="https://farm1.staticflickr.com/969/27198470517_b3501821ee_c.jpg"></a>
バスターミナル前の安宿に部屋を取り、町歩きに出かける。
バスターミナルの前に大きな市場がある。市場に食堂がたくさんある。
値段を見てみると寧南よりもさらに安い。夕食は市場で食べることにした。

<br>

<a href="https://farm1.staticflickr.com/945/42023916782_f781072f8e_o.jpg"><img alt="巧家, 3" src="https://farm1.staticflickr.com/945/42023916782_9be2f88d91_c.jpg"></a>
繁華そうな方に向かって歩く。

<br>

<a href="https://farm1.staticflickr.com/903/28197296248_09a70efbbc_o.jpg"><img alt="巧家, 4" src="https://farm1.staticflickr.com/903/28197296248_ef2d03e579_c.jpg"></a>
また市場があったのでのぞく。
そういえば今日も朝から何も食べていなかったので、市場で米線を食べる。
雲南省に来たら米線を食べるべきなのだ、きっと。

<br>

<a href="https://farm1.staticflickr.com/982/42025104632_3cf5366052_o.jpg"><img alt="巧家, 5" src="https://farm1.staticflickr.com/982/42025104632_b92c40086b_c.jpg"></a>
自分で好みの味に調整できるように、雲南の米線屋には調味料入れが置いてある。
唐辛子、山椒、塩に化学調味料。
開き直って恥ずかしげもなく化学調味料を置く雲南米線の図太さ。

<br>

<a href="https://farm1.staticflickr.com/950/42069221761_22cc7f4c26_o.jpg"><img alt="巧家, 6" src="https://farm1.staticflickr.com/950/42069221761_55c3845448_c.jpg"></a>
路地を進んで行く。

<br>

<a href="https://farm1.staticflickr.com/982/41169631045_9b70e571ea_o.jpg"><img alt="巧家, 7" src="https://farm1.staticflickr.com/982/41169631045_7238de1638_c.jpg"></a>
町の外れまで来てしまった。

<br>

<a href="https://farm1.staticflickr.com/969/41349187124_4bbf34b422_o.jpg"><img alt="巧家, 8" src="https://farm1.staticflickr.com/969/41349187124_9a9dd37c59_c.jpg"></a>
別の道から町に戻ると、また市場があった。
その後もぶらぶらと歩いているともうひとつ市場があった。
市場の多い町なのだな。

宿に戻り部屋でのんびりしていると、
ドアをノックされたので開けてみると公安と宿のお兄さんが立っていた。
外国人の泊まれるホテルに移れといわれるかと思ったら、
どこから来たのだ、どこへ行くのだ、旅行か仕事かとひと通り質問して、
ようこそ巧家へといって帰っていった。
宿のお兄さんもごめんね、公安に余計なこと言っちゃったよという感じで下りていった。

<br>

<a href="https://farm1.staticflickr.com/971/40261640970_17a27f0483_o.jpg"><img alt="巧家, 9" src="https://farm1.staticflickr.com/971/40261640970_4cde2d22ee_c.jpg"></a>
夕食は予定通り市場で、ナス炒めを食べた。

<br>

<a href="https://farm1.staticflickr.com/965/42069334871_5943ff95ea_o.jpg"><img alt="巧家, 10" src="https://farm1.staticflickr.com/965/42069334871_f049016301_c.jpg"></a>
翌日、バスで昭通に行く。昭通までは5時間もかかった。

<br>

<a href="https://farm1.staticflickr.com/956/41349743684_12a0f6d69d_o.jpg"><img alt="巧家, 11" src="https://farm1.staticflickr.com/956/41349743684_22f977ab65_c.jpg"></a>
バスターミナルはだいぶ郊外にあるようなので、市バスに乗って町中に出る。
宿が多くある場所でバスを降り、適当な安宿に部屋を取る。

<br>
