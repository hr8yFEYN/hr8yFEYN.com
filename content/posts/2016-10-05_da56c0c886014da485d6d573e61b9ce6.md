# 西昌古城
<a href="https://farm1.staticflickr.com/957/42023992182_4d6d36eed7_o.jpg"><img alt="西昌の古鎮をめぐる" src="https://farm1.staticflickr.com/957/42023992182_bd6cfe6f39_c.jpg"></a>
西昌には古鎮が残っているそうなので行ってみることにした。
清翠青年旅舎から1路のバスに乗り四牌楼で下車(<a href="https://goo.gl/maps/CTLY4BSbjjz">地図</a>)。

<br>

<a href="https://farm1.staticflickr.com/954/40261409110_481cecc9ef_o.jpg"><img alt="西昌の古鎮をめぐる, 1" src="https://farm1.staticflickr.com/954/40261409110_ce2c714e11_c.jpg"></a>
下りた交差点がもう古鎮のようで古い民家が多く並んでいる。
商店や食堂が並び賑わっている。

<br>

<a href="https://farm1.staticflickr.com/832/28196949588_ae2300051c_o.jpg"><img alt="西昌の古鎮をめぐる, 2" src="https://farm1.staticflickr.com/832/28196949588_4637f64fe2_c.jpg"></a>
観光地化されていないというだけあってボロい。

<br>

<a href="https://farm1.staticflickr.com/825/41348828414_9b7b41d40b_o.jpg"><img alt="西昌の古鎮をめぐる, 3" src="https://farm1.staticflickr.com/825/41348828414_d7af29f837_c.jpg"></a>
梁もだいぶつらそうだ。中国の古い民家は柱や梁が歪んでいる。
こういうのを見ると日本の建築技術の方がずっと優れていたのだなと思う。

<br>

<a href="https://farm1.staticflickr.com/908/41349573244_84854fe711_o.jpg"><img alt="西昌の古鎮をめぐる, 4" src="https://farm1.staticflickr.com/908/41349573244_d6dbc5f20f_c.jpg"></a>
昔ながらのホウロウのカップやら鍋やら。

<br>

<a href="https://farm1.staticflickr.com/962/40261668070_00a8f33279_o.jpg"><img alt="西昌の古鎮をめぐる, 5" src="https://farm1.staticflickr.com/962/40261668070_8b44e1118d_c.jpg"></a>
豆豉が多く売られていた。豆豉は日本でいう大徳寺納豆。…だと思う。

<br>

<a href="https://farm1.staticflickr.com/977/42069857391_1c2ee314e1_o.jpg"><img alt="西昌の古鎮をめぐる, 6" src="https://farm1.staticflickr.com/977/42069857391_c5752d24fa_c.jpg"></a>
お茶のんだり、白酒のんだり、タバコのんだり。

<br>

<a href="https://farm1.staticflickr.com/969/41349153414_8ec065125d_o.jpg"><img alt="西昌の古鎮をめぐる, 7" src="https://farm1.staticflickr.com/969/41349153414_4c715b2f0a_c.jpg"></a>
西昌はイ族が多いので、イ族の民族衣装を売る店が多い。

<br>

<a href="https://farm1.staticflickr.com/904/41350044124_c4a4038c63_o.jpg"><img alt="西昌の古鎮をめぐる, 8" src="https://farm1.staticflickr.com/904/41350044124_6b62a41d11_c.jpg"></a>
観光地化されていない古鎮ということだったが、来年には観光地化されるようだ。

<br>

# 礼州古鎮

<a href="https://farm1.staticflickr.com/910/42025177352_1c3b8a6bfe_o.jpg"><img alt="礼州古鎮" src="https://farm1.staticflickr.com/910/42025177352_72311ec0a1_c.jpg"></a>
西昌には郊外に礼州古鎮というこれもまた観光地化されていない古鎮があるというの行ってみた。
龍眼井街にある「大成」というバス停(<a href="https://goo.gl/maps/4APpVzDSxQq">地図</a>)から118路のバスに乗り、終点の礼州中学で下車(所要1時間程度、4元)。

<br>

<a href="https://farm1.staticflickr.com/910/40262073710_e5c255f7da_o.jpg"><img alt="礼州古鎮, 1" src="https://farm1.staticflickr.com/910/40262073710_9013e48ab1_c.jpg"></a>
バス停から西に向かうと礼州古鎮がある。

<br>

<a href="https://farm1.staticflickr.com/904/40261318890_a9a02742b4_o.jpg"><img alt="礼州古鎮, 2" src="https://farm1.staticflickr.com/904/40261318890_079354f711_c.jpg"></a>
門があり、門をくぐると古鎮に入るようだ。

<br>

<a href="https://farm1.staticflickr.com/963/41348779744_e86ce40cf7_o.jpg"><img alt="礼州古鎮, 3" src="https://farm1.staticflickr.com/963/41348779744_f55e110c6c_c.jpg"></a>
イ族隷属博物館で、このような梁がイ族の民家の特徴だと書いてあった。

<br>

<a href="https://farm1.staticflickr.com/954/40261241920_3dae598b1d_o.jpg"><img alt="礼州古鎮, 4" src="https://farm1.staticflickr.com/954/40261241920_3fc47623f0_c.jpg"></a>
お寺があった。いかにも中国という感じだ。

<br>

<a href="https://farm1.staticflickr.com/908/40262435990_d52e76aec2_o.jpg"><img alt="礼州古鎮, 5" src="https://farm1.staticflickr.com/908/40262435990_10fff6af06_c.jpg"></a>
いろいろ古い看板が残っている。

<br>

<a href="https://farm1.staticflickr.com/909/41168268185_cfb58c773a_o.jpg"><img alt="礼州古鎮, 6" src="https://farm1.staticflickr.com/909/41168268185_9c9d3dab2d_c.jpg"></a>
粉菓子のお店の看板。下の方に「各種誕生日のケーキ 予約受けたまります」と書いてある。

だいぶ古い看板のようだが、中国人はいつから誕生日にケーキを食べるようになったのか。
日本はいつから食べるようになったのか。そもそも誰が誕生日にケーキを食べることにしたのか。
キリストか、イエスか、不二家か。

<br>

<a href="https://farm1.staticflickr.com/905/27198667267_27275b9238_o.jpg"><img alt="礼州古鎮, 7" src="https://farm1.staticflickr.com/905/27198667267_1de2a57de9_c.jpg"></a>
セメントの建物も十分に古そうだった。

<br>
