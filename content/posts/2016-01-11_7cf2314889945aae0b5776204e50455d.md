 <a href="https://farm1.staticflickr.com/949/27198699387_c0c66d81f0_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き" src="https://farm1.staticflickr.com/949/27198699387_53280e5c6a_c.jpg"></a>
元陽(元阳)の定期市は日の干支が鼠と龍と猿の日に開かれる。9時前、宿を出て市を見に行く。今日も少し雲海が出ている。

<br>

<a href="https://farm1.staticflickr.com/911/40261703300_378457a610_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 1" src="https://farm1.staticflickr.com/911/40261703300_a796e7505c_c.jpg"></a>
まだ時間が早いようで準備中。どうも毎回毎回、マーケットに来るのが早すぎるのだ。

<br>

<a href="https://farm1.staticflickr.com/971/27199173347_fa34ace3dc_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 2" src="https://farm1.staticflickr.com/971/27199173347_56ac376064_c.jpg"></a>
下の市場周辺は多く露店が出ていたが人出は少ない。

<br>

<a href="https://farm1.staticflickr.com/972/41349769884_c9b0156c6f_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 3" src="https://farm1.staticflickr.com/972/41349769884_ac914f8791_c.jpg"></a>
お腹が空いていたので流行っている米線屋に入る。

<br>

<a href="https://farm1.staticflickr.com/912/41168870385_ac08be6f08_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 4" src="https://farm1.staticflickr.com/912/41168870385_863fa12a65_c.jpg"></a>
みんながワンタンを食べていたのでぼくも米線ではなくワンタンを食べる。

<br>

<a href="https://farm1.staticflickr.com/961/28196439438_85e3160951_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 5" src="https://farm1.staticflickr.com/961/28196439438_17c30ea120_c.jpg"></a>
ご飯を食べたので市場を見る。お肉屋さん。

<br>

<a href="https://farm1.staticflickr.com/954/27198562687_6651168097_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 6" src="https://farm1.staticflickr.com/954/27198562687_14417c31e2_c.jpg"></a>
野菜や果物屋さん。町の下にある市場の周辺は古い民家が残っている。

<br>

<a href="https://farm1.staticflickr.com/954/28195833028_543f597ed5_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 7" src="https://farm1.staticflickr.com/954/28195833028_ab836d17fd_c.jpg"></a>
元陽は山にある町なのでやはり階段や坂が多い。

<br>

<a href="https://farm1.staticflickr.com/951/27199696717_47857c95da_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 8" src="https://farm1.staticflickr.com/951/27199696717_9c39536fed_c.jpg"></a>
町の上の方に戻ると露店も多少開いているが、まだまだ人は少ない。

<br>

<a href="https://farm1.staticflickr.com/982/41350062594_06ff834235_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 9" src="https://farm1.staticflickr.com/982/41350062594_4ac3965862_c.jpg"></a>
ぶらぶらと町歩きをしてみる。路地に入る。

<br>

<a href="https://farm1.staticflickr.com/949/28196931928_158c7602f1_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 10" src="https://farm1.staticflickr.com/949/28196931928_0dd50a9bed_c.jpg"></a>
家のすぐ裏で野菜を育てている。とっても地産地消。

<br>

<a href="https://farm1.staticflickr.com/829/42024649052_e6a38600a8_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 11" src="https://farm1.staticflickr.com/829/42024649052_3389c7e286_c.jpg"></a>
レンガの家が多い。古い家もレンガ。新しい家もレンガ。

<br>

<a href="https://farm1.staticflickr.com/903/42069176291_1bf8085dff_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 12" src="https://farm1.staticflickr.com/903/42069176291_cd07d54712_c.jpg"></a>
古い家は少し曲線的。

<br>

<a href="https://farm1.staticflickr.com/975/41168781475_5968eeb900_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 13" src="https://farm1.staticflickr.com/975/41168781475_52e79e3ab3_c.jpg"></a>
新しい家はすごく直線的。

<br>

<a href="https://farm1.staticflickr.com/947/40261993820_9902158a64_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 14" src="https://farm1.staticflickr.com/947/40261993820_5728b537a7_c.jpg"></a>
家もいろいろ拡張される。住んでいるうちにいろいろつじつまの合わないことが出てきて、つじつまを合わせようと必死に拡張して、またつじつまが合わなくなって。結局、つじつまの合ってることなんてなくなった。

<br>

<a href="https://farm1.staticflickr.com/908/42024729652_c50e349d43_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 15" src="https://farm1.staticflickr.com/908/42024729652_3aac3066c9_c.jpg"></a>
階段を登って。


<br>

<a href="https://farm1.staticflickr.com/909/41349478114_9779cc0099_o.jpg"><img alt="元陽のマーケットとぶらぶら町歩き, 16" src="https://farm1.staticflickr.com/909/41349478114_a074178e63_c.jpg"></a>
町の中心に戻ると人出が多くなっていた。

<br>
