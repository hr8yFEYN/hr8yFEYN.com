 - <a href="/travel/2016/05/30/">ニャロン(新龍) 旅行情報</a>
- <a href="/travel/2016/06/11/">アチェンガル(亜丁) 旅行情報</a>

<br>

<a href="https://farm1.staticflickr.com/961/41168909545_8ab55aae6d_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ" src="https://farm1.staticflickr.com/961/41168909545_f9b6130fd1_c.jpg"></a>
朝7時から歩き始める。歩き始めるとすぐに集落が見えてくる。集落の後ろの山を越えると通宵に行くらしい。

<br>

<a href="https://farm1.staticflickr.com/825/28197365248_0ba3f16d6b_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 1" src="https://farm1.staticflickr.com/825/28197365248_c24873f670_c.jpg"></a>
集落を抜けると急な登りになる。

<br>

<a href="https://farm1.staticflickr.com/979/40262040750_eea63336af_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 2" src="https://farm1.staticflickr.com/979/40262040750_7af206b9b1_c.jpg"></a>
こんな道を歩くのは地元のチベット人くらいなもので、旅行だといってこんなところを歩いているぼくは不審過ぎるようで集落から監視隊が派遣された。

監視隊は通宵まで一緒に来るという。隊長は何やら電話でやり取りをしてる。きっと「今のところ不審な動きはなく、順調に通宵に向かっています。どうぞ」などと集落の人にいっているのだろう。

<br>

<a href="https://farm1.staticflickr.com/823/41169743215_877de76f5a_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 3" src="https://farm1.staticflickr.com/823/41169743215_d5fe27d6a1_c.jpg"></a>
しばらくすると監視隊が遅れ始める。せっかくなのでみんなで歌でも歌いながら行きますかといい、ひとり「仰げば尊し」を歌いながら歩いている外国人の相手などしたくなくなってきたようだ。

そのうちに本当に馬鹿馬鹿しくなったのか、集落のテリトリーを過ぎたのか帰っていってしまった。

<br>

<a href="https://farm1.staticflickr.com/956/42069956401_55574d1f16_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 4" src="https://farm1.staticflickr.com/956/42069956401_bd2a6703dd_c.jpg"></a>
監視隊と別れたあたりからバイクも通れないような道になる。

<br>

<a href="https://farm1.staticflickr.com/906/40262010600_19ae8f88cd_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 5" src="https://farm1.staticflickr.com/906/40262010600_a5105ff771_c.jpg"></a>
登ること1時間、10時に峠に着く。

<br>

<a href="https://farm1.staticflickr.com/953/41349840014_1b11af6ae6_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 6" src="https://farm1.staticflickr.com/953/41349840014_faaf7cda93_c.jpg"></a>
下の方に町が見える。あの村が通宵で、村から延びている立派な道がきっとアチェンガル・ゴンパに行く道だ。

<br>

<a href="https://farm1.staticflickr.com/974/40261917570_0861dca92e_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 7" src="https://farm1.staticflickr.com/974/40261917570_4c6fe08eac_c.jpg"></a>
松林の中をひたすら下る。去年峠を下っているときに膝を捻挫したので慎重に下っていく。

<br>

<a href="https://farm1.staticflickr.com/971/42024657132_87cf9960b1_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 8" src="https://farm1.staticflickr.com/971/42024657132_16b7506337_c.jpg"></a>

2時間ほど下ると通宵に着いた。感じのいい民家が多くある。

<br>

<a href="https://farm1.staticflickr.com/912/41349476614_6ac757a015_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 9" src="https://farm1.staticflickr.com/912/41349476614_bb82080bf2_c.jpg"></a>
町の中心には学校やら郷政府やら共産党らしい建物が多い。

商店があったので食料を少し買う。ザンパ(麦こがし)を買おうとしたらただでくれた。ありがたい有り難い。

<br>

<a href="https://farm1.staticflickr.com/947/40262574190_081c2c7d22_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 10" src="https://farm1.staticflickr.com/947/40262574190_04c3fd5a79_c.jpg"></a>
アチェンガル・ゴンパに向かう道は予想通り立派な舗装路。

<br>

<a href="https://farm1.staticflickr.com/946/42024345002_3c3120c943_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 11" src="https://farm1.staticflickr.com/946/42024345002_2215024ce4_c.jpg"></a>
町のはずれにゴンパがあったので見に行くも閉まっていて中は見れなかった。

<br>

<a href="https://farm1.staticflickr.com/953/28196827748_1940a09aa4_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 2日目 - 峠を越えて通宵へ, 12" src="https://farm1.staticflickr.com/953/28196827748_daa2ef9c4a_c.jpg"></a>
舗装路をだらだらと歩く。夕方、歩くのをやめ、テント泊。

<br>
