体調を崩しデルゲに1週間もいてしまったがマニカンコを目指して歩き始める。マニカンコまでは国道なのできっと楽な道のりだ。

8時、宿をチェックアウトして歩き始める。雨が降り出しそうな天気。

川沿いを歩く。採石のために川底をさらっているので川はにごっている。採石を終えると、 掘り起こした土砂などはそのままにして、別の場所に移って採石をしているようだ。

<br>

<a href="https://farm1.staticflickr.com/947/28196185268_a47332b25a_o.jpg"><img alt="デルゲからコラ山(雀児山), 1" src="https://farm1.staticflickr.com/947/28196185268_9d67e83c58_c.jpg"></a>
サキャ派の僧院が見えてくる。その近くに大仏があり中国人観光客はパシャパシャと写真を撮っている。他にも大きなマニ石などがありどうも中国人向けの観光チベットのようだ。

<br>

<a href="https://farm1.staticflickr.com/961/42025374432_5c39ac4b41_o.jpg"><img alt="デルゲからコラ山(雀児山), 2" src="https://farm1.staticflickr.com/961/42025374432_2ecde5eccc_c.jpg"></a>
昼過ぎ、コロという町に着く。ゴンパがあったので寄り道して見学。

<br>

<a href="https://farm1.staticflickr.com/967/41169543245_501b4d6317_o.jpg"><img alt="デルゲからコラ山(雀児山), 3" src="https://farm1.staticflickr.com/967/41169543245_7072ff2caf_c.jpg"></a>
サキャ派のゴンパでコロゴンパというらしい。こじんまりとしたいいお寺だ。馬もコルラさせられていた。

<br>

<a href="https://farm1.staticflickr.com/977/27198877577_75c5928ccc_o.jpg"><img alt="デルゲからコラ山(雀児山), 4" src="https://farm1.staticflickr.com/977/27198877577_3de7623ca1_c.jpg"></a>
新しいお堂を造っている。最近はコンクリートで造るところが多いようだが、木と土でトントンコンコンと造っている。

<br>

<a href="https://farm1.staticflickr.com/959/42068692531_5044b6ecf6_o.jpg"><img alt="デルゲからコラ山(雀児山), 5" src="https://farm1.staticflickr.com/959/42068692531_e992af4cd9_c.jpg"></a>
後ろに回り込んでみる。なんだかとってもいいもののように見える。

<br>

<a href="https://farm1.staticflickr.com/948/42024099352_0fd6432e85_o.jpg"><img alt="デルゲからコラ山(雀児山), 6" src="https://farm1.staticflickr.com/948/42024099352_b4bb3c994f_c.jpg"></a>
作業の邪魔かと思ったが、せっかくなので入らせてもらった。

<br>

<a href="https://farm1.staticflickr.com/908/41349093384_140ea26006_o.jpg"><img alt="デルゲからコラ山(雀児山), 7" src="https://farm1.staticflickr.com/908/41349093384_683e76cb0a_c.jpg"></a>
外では人手でトントンコンコンやっている。

いつかまた来て完成したゴンパを見てみたくなった。大きなコンクリートのゴンパよりもこういうゴンパに価値が出てくるとよさそうだけど、ラルンガルゴンパやアチェンガルゴンパなどでも大きなコンクリートのお堂がたくさん造られているのを見ると、チベット人には大きな伽藍の方がいいのかもしれない。

<br>

<a href="https://farm1.staticflickr.com/903/28196093658_6339542d7c_o.jpg"><img alt="デルゲからコラ山(雀児山), 8" src="https://farm1.staticflickr.com/903/28196093658_591271c687_c.jpg"></a>
コロを離れると雨が降り始める。夕方まで歩き、ゾクチェンゴンパに向かう道との分岐辺りでテント泊。

<br>

<a href="https://farm1.staticflickr.com/954/42069946091_3efd7be20a_o.jpg"><img alt="デルゲからコラ山(雀児山), 9" src="https://farm1.staticflickr.com/954/42069946091_719ed91955_c.jpg"></a>
2日目。明け方まで雨が降っていたが6時頃にやむ。青空が見えてきたので今日は晴れそうだ。

6時半から歩き始める。今日も国道をだらだらと歩く。

<br>

<a href="https://farm1.staticflickr.com/965/42070047101_277e765d67_o.jpg"><img alt="デルゲからコラ山(雀児山), 10" src="https://farm1.staticflickr.com/965/42070047101_98c8dd9cef_c.jpg"></a>
遠くに雪山が見えてくる。あれがコラ山かな。

<br>

<a href="https://farm1.staticflickr.com/824/28196729508_39ae39d5d5_o.jpg"><img alt="デルゲからコラ山(雀児山), 11" src="https://farm1.staticflickr.com/824/28196729508_9d9c33e439_c.jpg"></a>
ゴンパがあったので見学。サキャ派のゴンパでモンバゴンパというそうだ。本堂の規模の割に僧房が多い。

<br>

<a href="https://farm1.staticflickr.com/982/40262101780_0564547d58_o.jpg"><img alt="デルゲからコラ山(雀児山), 12" src="https://farm1.staticflickr.com/982/40262101780_b34dc47059_c.jpg"></a>
モンバゴンパを過ぎると雪山がだいぶ近くに見えてくる。

<br>

<a href="https://farm1.staticflickr.com/977/42068840571_9c86df9baf_o.jpg"><img alt="デルゲからコラ山(雀児山), 13" src="https://farm1.staticflickr.com/977/42068840571_481b772703_c.jpg"></a>
峠が見えてくる。国道はかなり大きく回り込んで登っていくようだ。細い道も見えるので、旧道を登って行けそうだ。

<br>

<a href="https://farm1.staticflickr.com/973/27199177667_6e1c35b54d_o.jpg"><img alt="デルゲからコラ山(雀児山), 14" src="https://farm1.staticflickr.com/973/27199177667_ee94b64aab_c.jpg"></a>
14時半から旧道を登り始める。思っていたほど急な登りではない。16時、テント泊。

<br>

<a href="https://farm1.staticflickr.com/958/27198897277_911de0c795_o.jpg"><img alt="デルゲからコラ山(雀児山), 15" src="https://farm1.staticflickr.com/958/27198897277_269fe54d75_c.jpg"></a>
3日目。朝6時半から登り始める。標高が4,600メートルあるので朝は寒い。

<br>

<a href="https://farm1.staticflickr.com/951/27198495907_2aa97dcee3_o.jpg"><img alt="デルゲからコラ山(雀児山), 16" src="https://farm1.staticflickr.com/951/27198495907_e326bf99a8_c.jpg"></a>
登り始めて1時間半、8時に国道と合流して峠に着く。

峠からコラ山が見えるのかと思っていたのたが、どうもこの登ってきた山がコラ山らしい。

峠に標識があり、標高5050メートルとあるが、そんなにないような気がする。バックパックからGPSを取り出し確認してみると4,890メートル。

標識には川蔵公路の最高地点と書かれているので、5,000メートル以上ということにしたかったのではないかという気がしてきた。だとしたら、とても中国らしい話だし、ぼくもとても納得できる。

標高が何メートルにせよ、後はマニカンコまで下るだけだ。
