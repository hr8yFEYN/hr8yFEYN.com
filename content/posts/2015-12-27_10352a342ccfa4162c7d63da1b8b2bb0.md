 <a href="https://farm1.staticflickr.com/905/28196194648_b74d7f7f50_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る" src="https://farm1.staticflickr.com/905/28196194648_fd0fe7fa27_c.jpg"></a>
十里村のマーケットを見たので、10時過ぎから金平に向けて歩き始める。十里村から金平までは13キロらしい。

歩き始めるとすぐに石敷きの道になる。

<br>

<a href="https://farm1.staticflickr.com/952/28196569108_5aeeefbab1_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 1" src="https://farm1.staticflickr.com/952/28196569108_07b523dd1d_c.jpg"></a>
天気がいいの歩いていても気持ちがいい。

<br>

<a href="https://farm1.staticflickr.com/943/42068551261_bdb5fb71ca_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 2" src="https://farm1.staticflickr.com/943/42068551261_d1e891f159_c.jpg"></a>
元陽のような立派な棚田ではないが、道は棚田の中を通って行く。

<br>

<a href="https://farm1.staticflickr.com/974/41168604285_47971b07ab_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 3" src="https://farm1.staticflickr.com/974/41168604285_04e26dab58_c.jpg"></a>
30分も歩くと遠くに金平の町が見えて来る。

<br>

<a href="https://farm1.staticflickr.com/904/42025169752_49e39b7783_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 4" src="https://farm1.staticflickr.com/904/42025169752_b45c5aa58d_c.jpg"></a>
白牛と黒牛。

<br>

<a href="https://farm1.staticflickr.com/952/41350102344_59545a3a85_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 5" src="https://farm1.staticflickr.com/952/41350102344_dbaf056319_c.jpg"></a>
1時間歩くと村に着く。

<br>

<a href="https://farm1.staticflickr.com/979/42023730832_effcffa526_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 6" src="https://farm1.staticflickr.com/979/42023730832_9c363ec953_c.jpg"></a>
川を渡る。


<br>

<a href="https://farm1.staticflickr.com/981/41168722625_8099651909_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 7" src="https://farm1.staticflickr.com/981/41168722625_67f4d8c367_c.jpg"></a>
しばらく歩くと金平の町がだいぶ近くに見えてくる。

<br>

<a href="https://farm1.staticflickr.com/829/40262588330_55940e64d9_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 8" src="https://farm1.staticflickr.com/829/40262588330_70e8e77402_c.jpg"></a>
金平の方に下りていく山道があったので下りて行ってみる。

<br>

<a href="https://farm1.staticflickr.com/906/42024968752_8d98c22279_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 9" src="https://farm1.staticflickr.com/906/42024968752_80bf9ec841_c.jpg"></a>
水路に出たので水路沿いに歩いて行く。

<br>

<a href="https://farm1.staticflickr.com/828/42024065312_f9b898445b_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 10" src="https://farm1.staticflickr.com/828/42024065312_7de5db9f67_c.jpg"></a>
順調に金平の町に近ずいている。

<br>

<a href="https://farm1.staticflickr.com/943/40261019950_84e8504f7d_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 11" src="https://farm1.staticflickr.com/943/40261019950_6a359c5e78_c.jpg"></a>
車道に合流した。

<br>

<a href="https://farm1.staticflickr.com/964/41168957695_bb45752fb4_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 12" src="https://farm1.staticflickr.com/964/41168957695_7fb9e4baff_c.jpg"></a>
あとは急斜面を下りていくと金平だ。

<br>

<a href="https://farm1.staticflickr.com/960/28196353138_708388b277_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 13" src="https://farm1.staticflickr.com/960/28196353138_5d7f77a90b_c.jpg"></a>
町中に入り階段を下りていく。

<br>

<a href="https://farm1.staticflickr.com/960/40262056430_bf524a1c22_o.jpg"><img alt="十里村から13キロ歩いて金平に戻る, 14" src="https://farm1.staticflickr.com/960/40262056430_b79492833c_c.jpg"></a>
階段を下りると十里村行きのミニバン乗り場に出た。

十里村から金平まで2時間半で着いた。
