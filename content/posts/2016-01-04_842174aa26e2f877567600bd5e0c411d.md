 <a href="https://farm1.staticflickr.com/967/41349274974_61e9446fa3_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き" src="https://farm1.staticflickr.com/967/41349274974_13d1bde0eb_c.jpg"></a>
8時半、朝食に出かける。昨日、夕食を食べた米線屋さんの水餃子がおいしそうだったので水餃子を食べに行く。水餃子の大椀を頼む(8元、160円)。ちょっと食べ過ぎたがおいしかった。

<br>

<a href="https://farm1.staticflickr.com/911/27199314027_be4c16aeb0_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 1" src="https://farm1.staticflickr.com/911/27199314027_54afbe22da_c.jpg"></a>
勐拉から者米に行くミニバンは昨晩泊まった全福賓館のすぐ隣に発着している。客待ちしている運転手に値段を聞くと25元(500円)だという。宿で聞くとバスもそろそろ来るころだというので急いでチェックアウトしてバスが来るのを待つ。

<br>

<a href="https://farm1.staticflickr.com/980/42069344951_7df9bba790_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 2" src="https://farm1.staticflickr.com/980/42069344951_223b32eaa7_c.jpg"></a>
30分ほど待つとバスが来る。バスのが少し安いのかと思ったが25元でミニバンと同じだった。翁当という町でマーケットが開かれていた。

<br>

<a href="https://farm1.staticflickr.com/955/42024204682_c8653d0e8c_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 3" src="https://farm1.staticflickr.com/955/42024204682_e2385cc60c_c.jpg"></a>
12時、者米に着く、勐拉からちょうど2時間。宿を数軒まわって値段を聞く、どこも50〜60元。紅福賓館の50元(1,000円)の部屋に決める。

<br>

<a href="https://farm1.staticflickr.com/943/42069148521_e99feb6981_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 4" src="https://farm1.staticflickr.com/943/42069148521_af5aae0cf7_c.jpg"></a>
宿も決まったので者米の町を歩きに行く。

<br>

<a href="https://farm1.staticflickr.com/952/40261884130_6eeabb2e68_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 5" src="https://farm1.staticflickr.com/952/40261884130_e66cb23d21_c.jpg"></a>
定期市が開かれる日以外は田舎の町だし静かだ。

<br>

<a href="https://farm1.staticflickr.com/824/28196902128_9134f535cf_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 6" src="https://farm1.staticflickr.com/824/28196902128_22b723ae9a_c.jpg"></a>
アールー族がいた。者米から老勐周辺はアールー族が多い。

<br>

<a href="https://farm1.staticflickr.com/979/28196380798_12f4b2936b_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 7" src="https://farm1.staticflickr.com/979/28196380798_c140948467_c.jpg"></a>
ベトナム人が路上で果物と雑貨を売っていた。片言のベトナム語で話しかけてみたらえらく笑われた。

<br>

<a href="https://farm1.staticflickr.com/947/41349290324_2072881d4f_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 8" src="https://farm1.staticflickr.com/947/41349290324_7d3234602c_c.jpg"></a>
市場も静かだ。

<br>

<a href="https://farm1.staticflickr.com/978/27198598727_c47d7c1534_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 9" src="https://farm1.staticflickr.com/978/27198598727_d0a318edee_c.jpg"></a>
市場の裏から町を出る道があったので進んでみる。どこか村にでも続くのかと思ったが川に出て行き止まりだった。

<br>

<a href="https://farm1.staticflickr.com/974/42068524611_cdae83ac87_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 10" src="https://farm1.staticflickr.com/974/42068524611_3a6b62d479_c.jpg"></a>
川の向こうに道があったので町まで戻り橋を渡る。

<br>


<a href="https://farm1.staticflickr.com/824/41350323474_c6983599c7_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 11" src="https://farm1.staticflickr.com/824/41350323474_8cff51e6a0_c.jpg"></a>
川沿いの道を進んでいく。

<br>

<a href="https://farm1.staticflickr.com/947/42069589981_8c229f20c8_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 12" src="https://farm1.staticflickr.com/947/42069589981_227bf40308_c.jpg"></a>
しばらく進むと小屋に出る。おじさんがいたのでこの道はどこに行きますかと聞くと、どこにも行かないという。

どこにも行かないというが折角なので少し歩いてみる。

<br>

<a href="https://farm1.staticflickr.com/962/41168434365_95c926780a_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 13" src="https://farm1.staticflickr.com/962/41168434365_b527d60b47_c.jpg"></a>
ゴムの樹があった。ゴムの樹を見ると南国に来たなという気がしてくるが、よくよく考えると寒い。ちっとも南国ではない。

<br>

<a href="https://farm1.staticflickr.com/970/42068819191_ef5c6112ff_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 14" src="https://farm1.staticflickr.com/970/42068819191_5eee246b9d_c.jpg"></a>
遠くに者米の町が見える。

<br>

<a href="https://farm1.staticflickr.com/950/42023718842_99d1182a46_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 15" src="https://farm1.staticflickr.com/950/42023718842_b69a72edb6_c.jpg"></a>
ずっと登って行くと車道に出た。

<br>

<a href="https://farm1.staticflickr.com/909/42069718961_7412a318e3_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 16" src="https://farm1.staticflickr.com/909/42069718961_0f689bb1d0_c.jpg"></a>
車道を歩く。

<br>

<a href="https://farm1.staticflickr.com/829/42025291392_2299b22fea_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 17" src="https://farm1.staticflickr.com/829/42025291392_a53f444b48_c.jpg"></a>
オレンジ色の村が出現する。ハニ族の村らしい。

外交部(日本でいう外務省)が援助をしたらしい。オレンジ色の建物は外交部の援助なのだ。

<br>

<a href="https://farm1.staticflickr.com/977/42024754812_ba422c0d64_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 18" src="https://farm1.staticflickr.com/977/42024754812_f22d2cb54f_c.jpg"></a>
村の入り口に山道があった。きっと登って来た山道に繋がっているのだろうとその山道を歩く。

<br>

<a href="https://farm1.staticflickr.com/954/41169421725_5c6094f9d7_o.jpg"><img alt="勐拉から者米に移動。者米をぶらぶら歩き, 19" src="https://farm1.staticflickr.com/954/41169421725_60499fe578_c.jpg"></a>
登りの道が車道と合流した場所に出た。あとは来た道を引き返して者米に戻る。

<br>
