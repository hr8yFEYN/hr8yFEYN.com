# イェンミンのマーケット

<a href="https://farm1.staticflickr.com/947/40262366760_0170f18828_o.jpg"><img alt="イェンミンのマーケットとザイ族の村" src="https://farm1.staticflickr.com/947/40262366760_a404bffa7e_c.jpg"></a>
イェンミンでは毎週日曜日にサンデーマーケットが開かれる。

昨日は民族衣装を着た人はあまり見かけなかったが今日は日曜市だけあって民族衣装を着た人が多い。

ドンバンではモン族が多く、僅かにザイ族とヤオ族がいるだけだったが、イェンミンのマーケットにはモン族ほどではないがザイ族とヤオ族もたくさんいる。

<br>

<a href="https://farm1.staticflickr.com/964/41348866304_aaff2c4ee1_o.jpg"><img alt="イェンミンのマーケットとザイ族の村, 1" src="https://farm1.staticflickr.com/964/41348866304_da8f2be006_c.jpg"></a>
やはりモン族の民族衣装を売る露店が多い。

<br>

<a href="https://farm1.staticflickr.com/944/42024669392_3141a60893_o.jpg"><img alt="イェンミンのマーケットとザイ族の村, 2" src="https://farm1.staticflickr.com/944/42024669392_3e00ddbfcb_c.jpg"></a>
晴れ着も売られている。

<br>

<a href="https://farm1.staticflickr.com/967/27199338547_538c3746e1_o.jpg"><img alt="イェンミンのマーケットとザイ族の村, 3" src="https://farm1.staticflickr.com/967/27199338547_92cdb940d3_c.jpg"></a>
ザイ族も多くいる。ザイ族は中国のプイ族(布依族)に近い民族なのかな。

<br>

<a href="https://farm1.staticflickr.com/910/28197077148_92acc2691e_o.jpg"><img alt="イェンミンのマーケットとザイ族の村, 4" src="https://farm1.staticflickr.com/910/28197077148_2c993acb22_c.jpg"></a>
ドンバンのヤオ族はアオザイヤオ族といわれるヤオ族だったようだが、イェンミンのヤオ族はランテンヤオ族のようだ。

<br>

<a href="https://farm1.staticflickr.com/903/28196047858_6fd5a4b521_o.jpg"><img alt="イェンミンのマーケットとザイ族の村, 5" src="https://farm1.staticflickr.com/903/28196047858_c510859911_c.jpg"></a>
マーケットの食堂でよく食べられている即席麺を食べてみる(8,000ドン、50円)。普通の即席麺だった。

<br>

# 少数民族の村歩き

<a href="https://farm1.staticflickr.com/953/27198588357_0466120aab_o.jpg"><img alt="少数民族の村歩き" src="https://farm1.staticflickr.com/953/27198588357_61a51e0fa0_c.jpg"></a>
イェンミンの近くにある少数民族の村を見に行く。できればヤオ族の村を見たいなと思いながら歩き始める。

<br>

<a href="https://farm1.staticflickr.com/951/41168762975_594a706584_o.jpg"><img alt="少数民族の村歩き, 1" src="https://farm1.staticflickr.com/951/41168762975_f1ed1b2a24_c.jpg"></a>
ヤオ族がマーケットから帰っていく方向に歩き始める。しばらく歩くと道は山を登って行く。

<br>

<a href="https://farm1.staticflickr.com/982/41169304405_9a1214d393_o.jpg"><img alt="少数民族の村歩き, 2" src="https://farm1.staticflickr.com/982/41169304405_d894764e12_c.jpg"></a>
山を登っていくと棚田が広がっている。

<br>

<a href="https://farm1.staticflickr.com/953/42024158182_c755057b1b_o.jpg"><img alt="少数民族の村歩き, 3" src="https://farm1.staticflickr.com/953/42024158182_297339a5a3_c.jpg"></a>
ザイ族の集落があった。高床式の家で、瓦で葺いた屋根がとっても立派。

<br>

<a href="https://farm1.staticflickr.com/824/28196006228_d24a146c98_o.jpg"><img alt="少数民族の村歩き, 4" src="https://farm1.staticflickr.com/824/28196006228_9da3b0dc68_c.jpg"></a>
霧が出てきて20メートル先も見えなくなる。それでもしばらく歩き続けるが視界が悪く村がどこにあるかわからないので下ることにする。

<br>

<a href="https://farm1.staticflickr.com/830/41169996385_0b5183a5ae_o.jpg"><img alt="少数民族の村歩き, 5" src="https://farm1.staticflickr.com/830/41169996385_02d4185994_c.jpg"></a>
下ると少し霧が晴れていて道から少し下ったところにザイ族の村があった。家はやはり高床式の家。

ヤオ族の村は結局見ることができなかったがザイ族の村を見れたので歩いたかいがあった。

<br>
