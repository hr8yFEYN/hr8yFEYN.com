 <a href="https://farm1.staticflickr.com/905/28196976388_d168e130e1_o.jpg"><img alt="アシュからセルシュ(石渠)" src="https://farm1.staticflickr.com/905/28196976388_628786e368_c.jpg"></a>
アシュ(阿須)からセルシュ(石渠)に向けて歩き始めるとすぐに分岐。多分どちらに行ってもセルシュに行くのだろうが、川沿いを行くよりも、ワシュ(瓦須)というところから山越えをして行った方が距離が短いので、左側のワシュに向う道に進む。

<br>

<a href="https://farm1.staticflickr.com/943/40261370620_d78b012657_o.jpg"><img alt="アシュからセルシュ(石渠), 1" src="https://farm1.staticflickr.com/943/40261370620_2bcab766d0_c.jpg"></a>
この辺は赤いタルチョが多い。一般的なカラフルなものよりかっこいい。

歩いているとランドクルーザーが止まる。お坊さんが大勢乗っていて、どこへ行くのだとかどこから来たのだとか、いつもの質問をされる。セルシュに行くというと、この先にあるサンガマニを見に行くといいと教えてくれた。

<br>

<a href="https://farm1.staticflickr.com/903/41169110225_22c26ce1df_o.jpg"><img alt="アシュからセルシュ(石渠), 2" src="https://farm1.staticflickr.com/903/41169110225_418ab11080_c.jpg"></a>
キツネを見かける。キツネと思ったが、イヌのような気もしてきた。ぼくはイヌもキツネもわからないということに気づく。オオカミといわれたらオオカミのような気もしてくる。イヌ、キツネ、オオカミというのは何が違うのか。

<br>

<a href="https://farm1.staticflickr.com/957/40261101750_e5783cf71f_o.jpg"><img alt="アシュからセルシュ(石渠), 3" src="https://farm1.staticflickr.com/957/40261101750_1eceb64f53_c.jpg"></a>
ウマ・ポニー・ロバ問題というのもありそうだ。

<br>

<a href="https://farm1.staticflickr.com/975/42023951422_83e66f44db_o.jpg"><img alt="アシュからセルシュ(石渠), 4" src="https://farm1.staticflickr.com/975/42023951422_342ff57a32_c.jpg"></a>
15時半、ワシュ(瓦須)に着く。ンゴツァゴンパというゴンパがあったので、ゴンパは食傷気味だが見学する。ワシュを過ぎるとしばらく町がないようなので、商店で少し多めに食料を買う。

<br>

<a href="https://farm1.staticflickr.com/951/28197168818_c9c1e3a5ff_o.jpg"><img alt="アシュからセルシュ(石渠), 5" src="https://farm1.staticflickr.com/951/28197168818_ff85ea2bf7_c.jpg"></a>
ワシュからしばらく歩きテント泊。夜、雷に雹、風も相当に強い。雷の落ちた音でテントがバリバリと振るえた。今回の旅行で一番の恐ろしさだった。

<br>

<a href="https://farm1.staticflickr.com/960/27199716407_4a7e1f6179_o.jpg"><img alt="アシュからセルシュ(石渠), 6" src="https://farm1.staticflickr.com/960/27199716407_dfda8a445c_c.jpg"></a>
2日目。小雨の中を歩く。

<br>

<a href="https://farm1.staticflickr.com/908/28195782318_dbdae56ab7_o.jpg"><img alt="アシュからセルシュ(石渠), 7" src="https://farm1.staticflickr.com/908/28195782318_e0e3e36b57_c.jpg"></a>
古い仏塔。ひとつだけセメントの新しい仏塔。

<br>

<a href="https://farm1.staticflickr.com/904/41349046034_54b5c4689d_o.jpg"><img alt="アシュからセルシュ(石渠), 8" src="https://farm1.staticflickr.com/904/41349046034_48ff7b87e7_c.jpg"></a>
峠が見えてくる。道はだいぶ大回りして登って行く。

<br>

<a href="https://farm1.staticflickr.com/981/40262350120_ef38dd26f1_o.jpg"><img alt="アシュからセルシュ(石渠), 9" src="https://farm1.staticflickr.com/981/40262350120_26b312aa94_c.jpg"></a>
旧道があったので、旧道を登る。

<br>

<a href="https://farm1.staticflickr.com/956/27199730457_1d5a4c35b0_o.jpg"><img alt="アシュからセルシュ(石渠), 10" src="https://farm1.staticflickr.com/956/27199730457_4ab112290d_c.jpg"></a>
1時間半ほど登ると峠に着く。

<br>

<a href="https://farm1.staticflickr.com/967/41348877574_821d5ffe7f_o.jpg"><img alt="アシュからセルシュ(石渠), 11" src="https://farm1.staticflickr.com/967/41348877574_60d2006f8a_c.jpg"></a>
峠を下るとまた草原。アシュ側に比べると人が少ない。

<br>

<a href="https://farm1.staticflickr.com/909/42068651591_55a649056f_o.jpg"><img alt="アシュからセルシュ(石渠), 12" src="https://farm1.staticflickr.com/909/42068651591_a66781ca51_c.jpg"></a>
夕方まで歩きテント泊。

<br>

<a href="https://farm1.staticflickr.com/957/41349176604_98ce1fdf18_o.jpg"><img alt="アシュからセルシュ(石渠), 13" src="https://farm1.staticflickr.com/957/41349176604_7388f3486c_c.jpg"></a>
3日目。朝から歩き、11時にアシュから川沿いを走る道と交流。この辺りは道路の工事をしているのかトラックやら作業員用のプレハブやらがたくさんある。プレハブの小さな商店が1軒だけあったので少し食料を買う。近くにチョムザゴンパというゴンパがあったがパスして先に進む。

<br>

<a href="https://farm1.staticflickr.com/907/27199000357_5a3254a28e_o.jpg"><img alt="アシュからセルシュ(石渠), 14" src="https://farm1.staticflickr.com/907/27199000357_0ae67fe70b_c.jpg"></a>
打ち捨てられていたような古い民家に新しい窓枠だとかドアの枠を付けていた。古民家再生だな。

<br>

<a href="https://farm1.staticflickr.com/823/42069236641_8098c52462_o.jpg"><img alt="アシュからセルシュ(石渠), 15" src="https://farm1.staticflickr.com/823/42069236641_92cebab253_c.jpg"></a>
夕方まで川沿いを歩きテント泊。夜はまた雷と雹。

<br>

<a href="https://farm1.staticflickr.com/826/41349834154_7277805941_o.jpg"><img alt="アシュからセルシュ(石渠), 16" src="https://farm1.staticflickr.com/826/41349834154_a215e9184e_c.jpg"></a>
4日目、昨日の雹で道はぬかるんでいて歩きにくい。

<br>

<a href="https://farm1.staticflickr.com/964/28196349128_d840e1c36c_o.jpg"><img alt="アシュからセルシュ(石渠), 17" src="https://farm1.staticflickr.com/964/28196349128_698be663d8_c.jpg"></a>
1時間も歩くと省道と合流。

<br>

<a href="https://farm1.staticflickr.com/828/41168951195_526a12df98_o.jpg"><img alt="アシュからセルシュ(石渠), 18" src="https://farm1.staticflickr.com/828/41168951195_c0c4c76e4b_c.jpg"></a>
立派な道。

<br>

<a href="https://farm1.staticflickr.com/949/42023960672_d6406cdfe8_o.jpg"><img alt="アシュからセルシュ(石渠), 19" src="https://farm1.staticflickr.com/949/42023960672_8894f20d2b_c.jpg"></a>
2時頃、町に着く。商店で買い物。


<br>

<a href="https://farm1.staticflickr.com/960/41168417325_6735e25fd4_o.jpg"><img alt="アシュからセルシュ(石渠), 20" src="https://farm1.staticflickr.com/960/41168417325_4a81ea8d1c_c.jpg"></a>
夕方まで歩き、セルシュまで13キロの場所でテント泊。


<br>

<a href="https://farm1.staticflickr.com/832/41169969615_f2b90744a3_o.jpg"><img alt="アシュからセルシュ(石渠), 21" src="https://farm1.staticflickr.com/832/41169969615_1a8f67fd02_c.jpg"></a>
4日目。歩き始めるとすぐにゴンパがある。ポニョゴンパというらしい。

<br>

<a href="https://farm1.staticflickr.com/982/42025123542_be2d950e78_o.jpg"><img alt="アシュからセルシュ(石渠), 22" src="https://farm1.staticflickr.com/982/42025123542_31448fcf56_c.jpg"></a>
9時、セルシュに着く。

<br>
