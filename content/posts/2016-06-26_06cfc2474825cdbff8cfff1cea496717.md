カトクゴンパからゾンサルゴンパ(宗薩寺)へはメチュ河を遡って行く。山越えなどがないのできっと楽な道。

昨日、カトクゴンパを見学した後、しばらく山を下ったところでテント泊をしたので、今日もまずは下っていく。

車道を下ったり、近道を下ったりしながらどんどん下っていく。

下り終えるとホルポまではほぼ完成している工事中の道。来年はカトクゴンパまで舗装路になっていそうだ。

<br>

<a href="https://farm1.staticflickr.com/826/27198627307_be7b0f5fda_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 1" src="https://farm1.staticflickr.com/826/27198627307_86b4bbd05e_c.jpg"></a>
道は川沿いを緩やかに下っていくのだが、何やら川の水を利用した装置がそこかしこにある。しかし何に使うものなのかさっぱりわからない。

<br>

<a href="https://farm1.staticflickr.com/970/40261224930_67d0cc2092_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 2" src="https://farm1.staticflickr.com/970/40261224930_bcea56537b_c.jpg"></a>
1時間半歩き、8時半にホルポ(河坡)に着く。商店や食堂が多くある。漢族の経営の店も多い。町にも漢族が多くいる。道路の工事やらカトクゴンパなどの工事で多くの漢族がこの辺りに働きに来ているのだろう。

<br>

<a href="https://farm1.staticflickr.com/958/41169004845_68d4c94db3_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 3" src="https://farm1.staticflickr.com/958/41169004845_5fbd7a28c8_c.jpg"></a>
ホルポからは未舗装路で車が通るたびに砂ぼこりがひどい。

<br>

<a href="https://farm1.staticflickr.com/946/41169851255_464a2fab17_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 4" src="https://farm1.staticflickr.com/946/41169851255_706ca16059_c.jpg"></a>
2時間も歩くとペユル(白玉)とデルゲ(徳格)の幹線道路に出る。

<br>

<a href="https://farm1.staticflickr.com/978/41169325495_b59c8849ef_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 5" src="https://farm1.staticflickr.com/978/41169325495_d0760814bf_c.jpg"></a>
河の対岸はチベット自治区だ、橋でもかかっていたら3分入域でもしてみようかなどと思いながら歩く。

<br>

<a href="https://farm1.staticflickr.com/968/42024532302_bfee7161fb_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 6" src="https://farm1.staticflickr.com/968/42024532302_bf3ceea8c1_c.jpg"></a>
13時、分岐に着く。トラックが来たので運転手さんに道を確認する。この道を進んで行き、道が分かれるところがあるので左に進めばゾンサルゴンパに行くらしい。運転手さんが、「じゃ、頑張れよ」とペットボトルの水をくれた。道を聞いて水までもらってしまった。

また車が通るたびに砂ぼこりの舞う未舗装路。

午後になって暑くなってきた。昨日、雪山を歩いていたとは思えないような暑さだ。

夕方まで暑いな暑いなと歩き、川辺にテントを張る。暑いので川で水浴びをしようと思ったが、水があまりに冷たく断念。世の中うまいこといかないものだ。

<br>

<a href="https://farm1.staticflickr.com/977/42024519832_2289a3e65a_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 7" src="https://farm1.staticflickr.com/977/42024519832_49e65af1fe_c.jpg"></a>
翌日、テントの撤収やパッキングに慣れて来たのか、朝6時半に出発の準備が整い、歩き始める。朝は少し肌寒い。

<br>

<a href="https://farm1.staticflickr.com/970/27198934027_74db5079bb_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 8" src="https://farm1.staticflickr.com/970/27198934027_4bf606bd88_c.jpg"></a>
歩き始めるとすぐに道が分かれる。昨日、トラックの運転手さんに教えてもらった通りに左に進む。

<br>

<a href="https://farm1.staticflickr.com/955/41170047725_c326350053_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 9" src="https://farm1.staticflickr.com/955/41170047725_3dea43a7fb_c.jpg"></a>
大きな滝があった。

<br>

<a href="https://farm1.staticflickr.com/970/28196292038_8b7f3b0a55_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 10" src="https://farm1.staticflickr.com/970/28196292038_d21c0204e0_c.jpg"></a>
滝を過ぎると景色が開けてくる。

11時頃になると民家が多くなる。そろそろ町がありそうだ。

岳巴という町に着く。町のはずれで昼食。たっぷり休んでから歩き始める。

<br>

<a href="https://farm1.staticflickr.com/963/42069415241_205736b6a6_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 11" src="https://farm1.staticflickr.com/963/42069415241_a3eb7006c9_c.jpg"></a>
この辺りは景色がいい。

<br>

<a href="https://farm1.staticflickr.com/964/41169933175_e5ae62026f_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 12" src="https://farm1.staticflickr.com/964/41169933175_1e63b21acb_c.jpg"></a>
また滝があった。

<br>

<a href="https://farm1.staticflickr.com/976/42068861561_4e3c68763e_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 13" src="https://farm1.staticflickr.com/976/42068861561_e83634da5d_c.jpg"></a>
道が西の方に曲がると遠くにゾンサルゴンパが見え始める。

<br>

<a href="https://farm1.staticflickr.com/957/42024599602_a63c197d41_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 14" src="https://farm1.staticflickr.com/957/42024599602_a933e5402a_c.jpg"></a>
14時半、ゾンサルゴンパに着く。ゾンサルゴンパはカトクゴンパと比べるとずっと落ち着いた雰囲気でいい。

<br>

<a href="https://farm1.staticflickr.com/970/41349921644_b4aae620c3_o.jpg"><img alt="カトクゴンパからゾンサルゴンパ(宗薩寺), 15" src="https://farm1.staticflickr.com/970/41349921644_7cdf2b7dfc_c.jpg"></a>
18時頃までのんびり見学して、ゴンパの向かいにある丘でテント泊。

<br>
