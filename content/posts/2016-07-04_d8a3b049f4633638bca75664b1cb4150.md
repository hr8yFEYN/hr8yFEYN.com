# アクセス・行き方
{% travel_info_item(img="https://farm1.staticflickr.com/824/41168488775_3a3b3d6b3f_c.jpg", link="https://farm1.staticflickr.com/824/41168488775_f2c4018a1d_o.jpg", alt="アクセス・行き方") %}

デルゲのバスターミナルは茶馬街と格薩尔大街の交差点から茶馬街を南に50メートル程行った左側にある(<a href="https://goo.gl/maps/hv4sPfasnem">地図</a>)。

{% end %}


## 康定
康定バスターミナルからデルゲ行きのバスが出ている。朝6:00発、所要11時間、190元。

デルゲから康定へはデルゲ・バスターミナルから朝6:00に出ている。所要11時間、174元。

## ペユル(白玉)
ペユルからデルゲへは、漢藏大酒店の前辺りでミニバンが客待ちをしている。60〜70元。

デルゲからペユルへはデルゲのバスターミナル前でミニバンが客待ちをしている。所要3時間、70元。

## マニカンコ(马尼干戈)
マニカンコからデルゲ行きのミニバンが出ている。70元、所要3時間。

マニカンコへは、デルゲのバスチケット売り場前でミニバンが客引きをしている。所要3時間、70元。

# 宿泊
旅館などの安宿が茶馬街に多くある。ドミトリーが20〜30元程度
、トイレ・シャワー共同のシングルやツインは50〜80元。WiFiが使えるところが多いが遅かったり、接続できないことも多いようなのでチェックイン前に確認した方がいい。

## 輪珠頂藏餐住宿
{% travel_info_item(img="https://farm1.staticflickr.com/908/27199297497_451e21a993_c.jpg", link="https://farm1.staticflickr.com/908/27199297497_fe942ba4eb_o.jpg", alt="輪珠頂藏餐住宿") %}

茶馬街と措拉街の交差点から茶馬街をマニカンコ方面に50メートル程度行った右側にある。1ベッド 30元。窓なしの狭いシングルルームもあり30元で泊まれる。

{% end %}


## 金徳賓館
{% travel_info_item(img="https://farm1.staticflickr.com/966/41349878854_5c82196b3f_c.jpg", link="https://farm1.staticflickr.com/966/41349878854_9aba4009c2_o.jpg", alt="金徳賓館") %}

市内から茶馬街をマニカンコ方面に行った右側にある。シングル・ツイン 50元。

{% end %}


## 風陵渡青年旅舎
{% travel_info_item(img="https://farm1.staticflickr.com/962/41169464335_3a81d639ce_c.jpg", link="https://farm1.staticflickr.com/962/41169464335_c4bf8cb0e8_o.jpg", alt="風陵渡青年旅舎") %}

金徳賓館からさらに500メートルほど先にある。ドミトリー 40元。ツイン 150元。

{% end %}


# 観光
{% travel_info_item(img="https://farm1.staticflickr.com/955/28196010888_61a0df7a4a_c.jpg", link="https://farm1.staticflickr.com/955/28196010888_9ebe8042d9_o.jpg", alt="観光") %}

デルゲ市内の見どころはデルゲゴチェンとデルゲパルカン。デルゲパルカンは拝観料がかかる(50元)。

デルゲ県にはペルプンゴンパ(八邦寺)、ゾンサルゴンパ(宗薩寺)、カトクゴンパ(噶蛇寺)と有名なゴンパがある。公共の交通機関はないのでヒッチハイクか、ミニバンのチャーターになる(言い値は500元)。各ゴンパの宿などの情報は<a href="/travel/2016/06/20/">ペユル旅行情報</a>を参照。

{% end %}
