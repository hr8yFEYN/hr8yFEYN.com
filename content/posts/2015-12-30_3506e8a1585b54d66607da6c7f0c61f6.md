 <a href="https://farm1.staticflickr.com/828/42024024272_ee07c22aa5_o.jpg"><img alt="金平から那発のマーケットを見に行く" src="https://farm1.staticflickr.com/828/42024024272_071329b11e_c.jpg"></a>
那発では日の干支が牛と羊の日に定期市が開かれる。金平から那発まではバスはなくミニバンでの移動になる。ミニバンは定員が集まらないと出発しなので早めに乗って待つことにする。7時に宿を出る。外はまだ暗い。南バスターミナル(金運客運駅)に行き那発行きのミニバンに乗る。客はまだぼくひとりだ。

<br>

<a href="https://farm1.staticflickr.com/954/42068553901_cb6921574a_o.jpg"><img alt="金平から那発のマーケットを見に行く, 1" src="https://farm1.staticflickr.com/954/42068553901_f74b32e84a_c.jpg"></a>
7時半過ぎに客がぼくを入れて4人になる。定員は7人なのであと3人。辺りはもう明るい。8時過ぎ、定員の7名になったので出発。金平から那発までも以前よりも道がずっとよくなっている。45分ほどで那発に着く(20元、400円)。

<br>

<a href="https://farm1.staticflickr.com/967/27198398447_e12e15bd19_o.jpg"><img alt="金平から那発のマーケットを見に行く, 2" src="https://farm1.staticflickr.com/967/27198398447_9439b7006f_c.jpg"></a>
バックパックを背負ってマーケットを見て回りたくないのでまずは宿を探す。2軒聞いてみるが60元(1,200円)。金平の宿が30元(600円)で安く済んだので60元でもよかったのだが、以前に何度か泊まっている迎賓旅社に行ってみる。トイレ・シャワー共同の20元(400円)だった部屋が30元になっていたが1泊なのでよしとする。

<br>

<a href="https://farm1.staticflickr.com/953/40262542260_ed3301f5f2_o.jpg"><img alt="金平から那発のマーケットを見に行く, 3" src="https://farm1.staticflickr.com/953/40262542260_8d761329e5_c.jpg"></a>
宿のおばさんに公安に行って登記して来てといわれたので部屋に荷物を置かせてもらい公安に行く。宿のおばさんがすでに公安に電話で連絡してくれていたようでスムースに手続きできる。パスポートのコピーを取られて、どこから来たのか、どこに行くのか、那発には何泊するのかと聞かれておしまい。

<br>

<a href="https://farm1.staticflickr.com/943/42025069182_54ee26ec46_o.jpg"><img alt="金平から那発のマーケットを見に行く, 4" src="https://farm1.staticflickr.com/943/42025069182_878baa875b_c.jpg"></a>
宿に戻り少し休憩してからマーケットを見に行く。道端には沙ヤオ族がすでに大勢いる。昔は衣装の縁取りはピンクだったのだが、最近は蛍光色のようなオレンジが流行りのようでみんなオレンジの縁取りの衣装を着ている。

<br>

<a href="https://farm1.staticflickr.com/911/41349464174_9216aaa781_o.jpg"><img alt="金平から那発のマーケットを見に行く, 5" src="https://farm1.staticflickr.com/911/41349464174_2d37a9c0d0_c.jpg"></a>
10過ぎなのだが、まだ少し早いのか人がマーケットの規模の割に少ない。

<br>

<a href="https://farm1.staticflickr.com/831/42024652562_954721cc20_o.jpg"><img alt="金平から那発のマーケットを見に行く, 6" src="https://farm1.staticflickr.com/831/42024652562_ed80d3136e_c.jpg"></a>
那発ではよく見かける民族。那発の他は勐拉で少し見かける程度。ラフ族なのかと思ったがハニ族らしい。ハニ族といってもアカ族などに近いのかもしれない。

<br>

<a href="https://farm1.staticflickr.com/825/42068711481_05876ae49f_o.jpg"><img alt="金平から那発のマーケットを見に行く, 7" src="https://farm1.staticflickr.com/825/42068711481_9c184e7c31_c.jpg"></a>
民族衣装を売るミャオ族(モン族)。民族衣装を売っている人はどこのマーケットでも着飾っている。自分で売り子とマネキンのひとりふた役をこなすのだ。

<br>

<a href="https://farm1.staticflickr.com/909/42024875012_627f0a072a_o.jpg"><img alt="金平から那発のマーケットを見に行く, 8" src="https://farm1.staticflickr.com/909/42024875012_755ffa594d_c.jpg"></a>
ベトナム人がベトナムの雑貨を売りに来ている。

<br>

<a href="https://farm1.staticflickr.com/912/40262411220_7698929a4d_o.jpg"><img alt="金平から那発のマーケットを見に行く, 9" src="https://farm1.staticflickr.com/912/40262411220_92aaf20efb_c.jpg"></a>
ベトナムの雑貨。

<br>

<a href="https://farm1.staticflickr.com/906/41168778615_fa49ddeba7_o.jpg"><img alt="金平から那発のマーケットを見に行く, 10" src="https://farm1.staticflickr.com/906/41168778615_d2212ff5b3_c.jpg"></a>
お米なんかも売りに来ている。ここだけ見たら中国ではなくベトナムだ。

<br>

<a href="https://farm1.staticflickr.com/981/41350120274_0e40144349_o.jpg"><img alt="金平から那発のマーケットを見に行く, 11" src="https://farm1.staticflickr.com/981/41350120274_f5d10278c8_c.jpg"></a>
ヤオ族。袖に刺繍をしている人もいる。ベトナムで見たアオザイヤオ族に近いのかな。

<br>

<a href="https://farm1.staticflickr.com/911/40262005500_7c1c6448be_o.jpg"><img alt="金平から那発のマーケットを見に行く, 12" src="https://farm1.staticflickr.com/911/40262005500_6b4ce62721_c.jpg"></a>
国境の方に歩いて行ってみる。沙ヤオ族がたくさんベトナムから来ている。昔は川の浅瀬を渡って来る人も多かったが、今回は川を渡って来る人は見かけなかった。少数民族もちゃんと出入国の手続きをしないといけなくなったのだろう。
